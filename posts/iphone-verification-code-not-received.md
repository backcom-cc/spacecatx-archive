# 아이폰 인증 문자만 안 옴 해결법(은행, 인스타, 이중인증 문자)

### 아이폰 인증 문자 안옴. 유독 인증 문자만 안오더라구요.

아이폰 15로 바꿨을 때 얘기예요.

설레는 마음으로 은행 앱이랑 인스타그램에 로그인하려는데 인증번호가 안 오는 거예요. 처음엔 그냥 늦게 오나 보다 했죠. 그런데 5분이 지나도, 10분이 지나도 감감무소식이었습니다.

지인한테 테스트 문자를 보내달라고 했더니 그건 1초 만에 왔어요. 문자 수신은 되는데 인증번호만 딱 안 오는 거예요. 모르는 번호 차단도 꺼뒀는데도요. 아악,, 뽑기 실패인가 싶어서 진짜 머리가 아프더라고요. 전 폰은 이미 끊겼는데, 교환받으러 가면 그 사이엔 또 어떻게 하나 싶고요.

알고 보니 원인이 한 가지가 아니었습니다. 통신사 설정부터 아이폰 내부 설정까지 체크해야 할 포인트가 몇 가지 있거든요. 이 글에서 그 순서대로 정리해드릴게요.

> **한줄 결론:** 새 아이폰으로 바꾼 뒤 인증 문자만 안 온다면, 기기 불량보다 메시지 필터와 통신사 차단 설정부터 확인하세요.

---

## 1. 메시지 앱에서 ‘알 수 없는 발신자’ 필터를 먼저 확인하세요

인증 문자가 안 올 때 제일 먼저 볼 곳은 메시지 앱이에요.

아예 안 온 게 아니라, 다른 탭에 따로 분류돼 있을 수 있거든요. 아이폰은 연락처에 없는 번호에서 온 문자를 자동으로 걸러주는 기능이 있는데, 은행 인증번호나 인스타그램 인증번호처럼 모르는 번호에서 오는 문자가 여기 들어갈 수 있습니다.

메시지 앱을 열면 상단에 필터 메뉴가 있습니다. 거기서 **알 수 없는 발신자**, **스팸**, **정크** 항목을 한번 확인해보세요.

아예 이 기능이 켜져 있는지 보려면 아래 경로로 들어가보세요.

```
설정 > 앱 > 메시지 > 알 수 없는 발신자 필터링
```

또는

```
설정 > 메시지 > 알 수 없는 발신자 필터링
```

켜져 있으면 일단 꺼두고 인증번호를 다시 요청해보세요. 설정을 바꾼 직후에는 바로 안 뜰 수 있으니, 잠깐 기다렸다가 메시지 앱을 다시 열어보는 게 좋습니다.

아이폰의 메시지 필터 기능은 애플 공식 도움말에서도 확인할 수 있습니다. 알 수 없는 발신자 필터가 어떻게 동작하는지 더 보고 싶다면 [애플 공식 안내](https://support.apple.com/guide/iphone/screen-and-filter-texts-iph203ab0be4/ios)를 참고해보세요.

---

## 2. 통신사 스팸 차단 설정을 확인하세요

메시지 앱에도 없다면 이번에는 통신사 쪽을 봐야 합니다.

아이폰 안에서는 아무리 찾아도 없는데, 실제로는 문자가 아이폰에 도착하기도 전에 통신사에서 걸러지는 경우가 있어요. 특히 일반 문자는 잘 오는데 인증번호만 안 온다면 여기가 원인일 가능성이 높습니다.

통신사 앱에서 스팸 차단 설정을 확인해보세요.

```
SKT: T월드 > 스팸 차단 관련 설정
```

```
KT: 마이케이티 > 스팸 차단 관련 설정
```

```
LG U+: 당신의 U+ > 스팸 차단 관련 설정
```

```
알뜰폰: 사용 중인 통신사 고객센터 또는 앱
```

은행이나 본인인증 문자는 15xx, 16xx, 18xx 같은 번호로 오는 경우가 많습니다. 이런 번호가 스팸으로 잡혀 있으면 일반 문자는 오는데 인증번호만 쏙 빠지는 상황이 생길 수 있어요.

앱에서 찾기 어렵다면 고객센터에 문의하는 게 빠를 수 있습니다.

---

## 3. 인증번호를 너무 여러 번 눌렀다면 잠깐 멈추세요

안 오면 다시 보내기 버튼을 계속 누르게 되는데, 이게 오히려 역효과일 수 있습니다.

너무 짧은 시간 안에 여러 번 요청하면 서비스 쪽에서 발송을 잠깐 막아버릴 수 있거든요. 은행, 인스타그램, 구글 이중인증처럼 보안이 들어간 서비스는 특히 그렇습니다. 이 경우에는 아이폰 문제가 아니라 인증번호 요청 자체가 제한된 상태일 수 있어요.

이럴 땐 그냥 멈추는 게 제일 빠릅니다. 물 한 잔 마시고 5~10분 기다렸다가, 앱을 완전히 종료하고 다시 열어서 한 번만 요청해보세요.

---

## 4. 아이폰을 재시작한 뒤 다시 요청해보세요

여기까지 해봤는데도 안 온다면 아이폰 재시작을 한 번 해보세요.

초기화가 아닙니다. 일시적으로 엉킨 네트워크 연결이나 메시지 앱 상태를 새로고침하는 거예요. 껐다 켜는 것만으로도 해결되는 경우가 생각보다 있습니다.

재시작하고 나서 바로 여러 번 누르지 말고, 앞에서 많이 요청했다면 5분 정도 기다렸다가 한 번만 다시 요청해보세요.

---

## 5. 그래도 안 되면 네트워크 설정 재설정

이것저것 다 해봤는데도 계속 안 온다면 마지막으로 네트워크 설정을 재설정해보세요.

아이폰 전체 초기화가 아니라, 와이파이와 셀룰러 연결 설정만 다시 잡는 방법입니다. 네트워크 설정이 꼬여서 문자 수신이 제대로 안 되는 경우에 효과가 있을 수 있어요.

단, 저장된 와이파이 비밀번호가 지워지니까 마지막 수단으로 두세요.

```
설정 > 일반 > 전송 또는 iPhone 재설정 > 재설정 > 네트워크 설정 재설정
```

재설정 후 아이폰이 다시 켜지면 와이파이를 연결하고, 인증번호를 한 번만 다시 요청해보세요.

---

## 아이폰 인증 문자 안옴 마치며

저는 새 폰으로 바꾸면서 초기 설정이 덜 잡힌 것과 통신사 차단이 겹친 경우였는데, 주변 얘기 들어보면 생각보다 메시지 앱 필터에 걸려서 놓치는 경우가 제일 많더라고요.

인증번호 안 온다고 당황하기 전에 필터 탭부터 한번 확인해보세요. 일반 문자는 오는데 인증번호만 안 온다면, 아이폰 고장보다 메시지 필터와 통신사 차단 설정을 먼저 보는 게 좋습니다.
