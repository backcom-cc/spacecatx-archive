## 구글 AI 요약, 편하긴 한데 가끔은 원문이 더 보고 싶죠? 구글 AI 요약 없이 검색 결과만 보는 법

요즘 구글 검색을 하면 맨 위에 AI가 요약해주는 화면이 뜰 때가 있습니다. 빠르게 훑어보기엔 편하지만, 자료 조사나 정확한 확인이 필요할 때는 AI 요약보다 원문 링크를 바로 보고 싶을 때가 있어요.

특히 논문, 법률, 의학, 기술 자료처럼 출처 확인이 중요한 검색에서는 요약보다 웹 결과가 더 유용할 수 있습니다. “AI가 틀렸다”는 뜻이 아니라, 내가 직접 원문을 보고 판단하고 싶은 상황인 거죠.

이럴 때는 구글 AI 요약 없이 웹 결과만 보는 방법을 알아두면 꽤 편합니다.

---

## 구글 AI 요약 없이 웹 결과만 보는 방법

구글 검색 설정에서 AI 요약을 완전히 끄는 공식 스위치가 따로 있는 것은 아닙니다. 대신 검색 결과의 **Web** 탭을 직접 열거나, 검색 주소에 `&udm=14`를 붙여 웹 결과 중심 화면으로 볼 수 있습니다.

가장 간단한 방식은 검색 결과 주소 뒤에 아래 코드를 붙이는 것입니다.

```
&udm=14
```

예를 들어 이렇게 검색할 수 있습니다.

```
https://www.google.com/search?q=검색어&udm=14
```

이렇게 접속하면 AI 요약이나 일부 확장 영역이 줄어든, 웹 링크 중심의 결과 화면을 볼 수 있습니다.

---

## 검색 결과에서 Web 탭을 눌러도 됩니다

주소 뒤에 코드를 붙이는 것이 번거롭다면, 검색 결과 화면에서 **Web** 또는 **웹** 탭을 누르는 방법도 있습니다. 이미지, 뉴스, 동영상 탭이 있는 줄에서 Web 탭을 선택하면 AI 요약이나 확장 요소가 줄어든 웹 링크 중심 결과를 볼 수 있습니다.

Web 탭이 바로 보이지 않는다면 **더보기** 메뉴 안에 들어가 있을 수 있습니다. 이 방법은 검색할 때마다 한 번씩 눌러야 하지만, 별도 설정 없이 바로 쓸 수 있다는 장점이 있습니다.

가끔만 웹 결과만 보고 싶다면 이 방법이 가장 간단합니다. 매번 AI 요약이 불편한 것은 아니고, 특정 검색에서만 원문 링크를 빠르게 확인하고 싶다면 Web 탭을 누르는 방식이면 충분합니다.

---

## 매번 &udm=14 붙이기 귀찮다면 주소창 검색으로 설정하세요

매번 검색 주소 뒤에 `&udm=14`를 붙이는 건 귀찮습니다. 자주 쓸 거라면 크롬 주소창 검색 엔진에 웹 결과 전용 검색을 추가해두는 방법이 더 편합니다.

한 번만 설정해두면 주소창에서 검색할 때 AI 요약이 없는 웹 결과 중심 화면으로 바로 이동할 수 있습니다.

### 크롬에서 구글 웹 결과 검색 추가하는 방법

1. 크롬 오른쪽 위 점 3개 메뉴를 누릅니다.
2. **설정**으로 들어갑니다.
3. **검색 엔진**을 선택합니다.
4. **검색 엔진 및 사이트 검색 관리**로 들어갑니다.
5. **사이트 검색** 항목에서 **추가**를 누릅니다.
6. 아래 내용을 입력합니다.

```
검색 엔진: 구글 웹 결과
바로가기: g
URL: https://www.google.com/search?q=%s&udm=14
```

이렇게 추가한 뒤 주소창에 `g 검색어`처럼 입력하면, 해당 검색어가 웹 결과 중심으로 열립니다. 예를 들어 `g 아이폰 배터리 교체`처럼 입력하는 방식입니다.

만약 이 검색을 항상 기본으로 쓰고 싶다면, 추가한 검색 엔진 옆의 점 3개 메뉴에서 기본으로 설정할 수 있습니다. 다만 크롬 버전이나 환경에 따라 기본 검색 엔진으로 설정 가능한 항목이 다르게 보일 수 있으니, 안 된다면 바로가기 방식으로만 사용해도 충분합니다.

---

## 이게 우회인가요?

보안 취약점을 이용하는 방식은 아닙니다. `udm=14`는 구글 검색 결과 표시 방식을 바꾸는 URL 파라미터로 알려져 있습니다. 쉽게 말하면 검색 결과를 웹 링크 중심으로 보는 표시 옵션에 가깝습니다.

다만 이것은 “구글 AI 요약을 영구적으로 끄는 공식 설정”이라기보다는, 검색 결과를 Web 모드로 여는 방법에 가깝게 이해하는 것이 좋습니다. 구글 화면 구성은 언제든 바뀔 수 있으니, 나중에 동작 방식이 달라질 수 있다는 점도 알아두세요.

관련 내용은 Ars Technica의 [Google Search’s “udm=14” trick lets you kill AI search for good](https://arstechnica.com/gadgets/2024/05/google-searchs-udm14-trick-lets-you-kill-ai-search-for-good/) 기사에서도 소개된 바 있습니다.

## 이런 분들에게 유용합니다

AI 요약이 무조건 나쁜 것은 아닙니다. 빠르게 개요를 볼 때는 편합니다. 하지만 원문 출처를 직접 확인해야 하는 검색이라면 웹 결과만 보는 편이 더 나을 수 있습니다.

* AI 요약보다 직접 링크를 빠르게 보고 싶은 분
* 논문, 기술 문서, 공식 도움말을 찾는 분
* 법률, 의료, 금융처럼 정확성이 중요한 정보를 확인하는 분
* 블로그 글감이나 SEO 검색 결과를 직접 비교하는 분
* 검색 화면이 복잡하게 느껴지는 분
* 원문 링크와 게시 날짜를 먼저 확인하고 싶은 분

특히 자료 조사나 블로그 키워드 확인을 할 때는 AI 요약보다 실제 상위 문서가 무엇인지 직접 보는 것이 더 중요할 때가 많습니다.

## 즐겨찾기로 저장해두는 방법도 있습니다

자주 쓰는 검색 형태가 있다면 `&udm=14`가 붙은 주소를 즐겨찾기로 저장해두는 것도 방법입니다. 예를 들어 특정 사이트 안에서만 검색하고 싶다면 아래처럼 저장해둘 수 있습니다.

```
https://www.google.com/search?q=site:example.com&udm=14
```

이 주소를 즐겨찾기해두면 필요할 때 웹 결과 중심으로 바로 접근할 수 있습니다. 다만 검색어를 매번 바꿔가며 자주 쓸 예정이라면, 앞에서 설명한 크롬 주소창 검색 엔진 등록 방식이 더 편합니다.

---

## 정리

구글 AI 요약은 빠르게 개요를 볼 때 편리하지만, 원문 확인이 중요한 검색에서는 웹 결과 중심 화면이 더 편할 수 있습니다.

**구글 AI 요약 없이 웹 결과만 보고 싶다면 검색 결과에서 Web 탭을 누르거나, 검색 주소에 `&udm=14`를 붙이고, 자주 쓴다면 크롬 사이트 검색으로 등록해두세요.**
