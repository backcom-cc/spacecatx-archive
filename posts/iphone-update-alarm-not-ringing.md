### 아이폰 업데이트 후 알람 안 울림,, 덕분에 백수 될 뻔하다…

아니, 업데이트 하고 나서 알람이 간헐적으로 안 울리는 거 저만 그런가요?

저는 원래 아침마다 못 일어날까봐 10분 단위로 알람을 서너 개씩 겹쳐서 맞춰놓는 편이었어요. 그런데 어느 날 눈을 떴는데 분위기가 이상한 거예요. 시계를 보니 마지막 알람 시각이 한참 지나 있었고, 알고 보니 앞에 맞춰놓은 알람들은 다 조용히 넘어가고 제일 마지막 것만 딸랑 울렸더라고요.

그날 거의 미친 듯이 뛰어나가서 지각을 겨우 면했는데, 며칠 뒤에 똑같은 일이 또 생기니까 이러다 진짜 잘릴 수도 있겠다 싶었어요. 백수 되기 일보 직전이었습니다. 알람을 여러 개 켜놓는 의미 자체가 없어지는 거라, 이건 그냥 넘어갈 일이 아니다 싶어서 제대로 한번 파봤어요.

> **한줄결론:**   
> 아이폰 업데이트 후 알람이 간헐적으로 안 울린다면, 범인은 알람 자체보다 수면 집중 모드와 기상 알람이 겹친 ‘알람 주변 설정’일 가능성이 큽니다.

## 알람이 사실 하나가 아니었다는 거, 알고 계셨나요

아이폰 알람이 헷갈리는 이유 중 하나는, 우리가 생각하는 “알람”이 사실 하나가 아니라서예요.

시계 앱에 직접 맞춰둔 알람이 있고, 건강 앱 수면 일정에 연결된 기상 알람이 따로 있어요. 여기에 수면 집중 모드까지 자동으로 켜졌다 꺼졌다 하면, 뭐가 먼저 동작하고 뭐가 조용히 넘어간 건지 헷갈릴 수밖에 없죠. 알람을 여러 개 겹쳐 맞춰두는 분이라면 더더욱요.

먼저 건강 앱 수면 일정부터 확인해보세요.

```
건강 앱 > 둘러보기 > 수면 > 전체 일정 및 옵션
```

기상 시간이 어떻게 잡혀 있는지, 기상 알람이 켜져 있는지 확인합니다.

그리고 수면 집중 모드도 같이 체크하세요.

```
설정 > 집중 모드 > 수면
```

수면 집중 모드가 자동으로 켜지고 꺼지는 시각을 확인해보세요. 내가 생각한 기상 시간이랑 실제 수면 모드 종료 시간이 다르면, 알람이 이상하게 느껴질 수 있어요.

업데이트 후 알람이 영 불안정하다면, 하루 정도는 구조를 단순하게 만들어보는 것도 좋아요.

```
수면 기상 알람 잠깐 끄기
시계 앱 알람만 켜기
다음 날 정상적으로 울리는지 확인
```

이렇게 하면 문제가 수면 일정 쪽인지, 시계 앱 알람 자체인지 훨씬 명확하게 구분됩니다.

## 아이폰 업데이트 알람 안 울림 급하면 이것부터, 알람 직전에 수면 집중 모드 꺼버리기

내일 아침이 진짜 중요한 날이라면, 설정 하나하나 뒤지기 전에 이것부터 해보세요. 원래는 수면 집중 모드가 켜져 있어도 시계 앱 알람은 울리는 게 정상입니다. 그런데 업데이트 후에는 이 부분이 애매하게 꼬여서, 알람이 막히거나 조용해지는 것처럼 느껴지는 경우가 있더라고요.

그래서 아예 알람이 울리기 직전에 수면 집중 모드를 자동으로 꺼버려서, 알람이 울릴 때는 집중 모드 영향을 받을 일 자체를 줄이는 방법입니다.

단축어 앱에서 자동화를 하나 만들면 됩니다.

```
단축어 앱 > 자동화 > + 버튼
```

여기서 **시간**을 선택하는데, 평소 알람보다 1분 정도 빠르게 잡는 게 포인트예요. 알람을 오전 7시에 맞춰뒀다면 자동화는 오전 6시 59분으로요.

```
오전 6:59 수면 집중 모드 자동 해제
오전 7:00 알람 울림
```

반복은 **매일**로 설정하고, 동작 추가에서 아래 항목을 찾아주세요.

```
동작 추가 > 집중 모드 설정 > 수면 끄기
```

마지막으로 **실행 전에 묻기** 옵션은 꼭 꺼두세요. 켜져 있으면 자동화 직전에 확인 알림이 뜨는데, 자고 있는 동안엔 그걸 볼 수가 없으니까 정작 자동화가 실행도 안 되고 넘어갈 수 있어요.

이게 모든 알람 오류를 해결해주는 만능 해법은 아니에요. 정확히 왜 먹히는지까지 검증된 방법은 아니지만, 수면 집중 모드를 쓰고 있고 업데이트 후 알람이 간헐적으로 안 울린다면 가장 먼저 시도해볼 만한 우회법입니다.

저처럼 지각이 걱정이라면, 이걸로 일단 해보세요. 직장은 소중해요ㅠㅠ

## 업데이트 직후라면 강제 재시동도 해보세요

위에 다 확인했는데도 여전히 불안하다면, 강제 재시동으로 한 번 정리해보세요. 업데이트 직후에는 알람뿐 아니라 알림, 배터리, 발열, 키보드 반응까지 전체적으로 어색하게 느껴질 때가 있습니다.

```
볼륨 올리기 버튼 짧게 누르기
볼륨 내리기 버튼 짧게 누르기
애플 로고가 나올 때까지 측면 버튼 길게 누르기
```

강제 재시동 방법은 모델별로 차이가 있을 수 있으니, 버튼 순서가 헷갈린다면 애플 공식 안내도 같이 확인해보세요.

[Apple 공식 지원: iPhone 강제로 재시동하기](https://support.apple.com/ko-kr/guide/iphone/iph8903c3ee6/ios)

## 소리가 너무 작아서 못 들었을 수도 있어요

알람이 아예 안 울린 줄 알았는데, 사실은 소리가 너무 작아서 못 들은 경우도 있어요. 아이폰은 미디어 음량이랑 벨소리 및 알림 음량이 따로 움직이거든요. 유튜브나 음악 소리는 줄였어도 알람 쪽 음량은 따로 낮아져 있을 수 있는 거죠.

```
설정 > 사운드 및 햅틱 > 벨소리 및 알림 음량
```

슬라이더가 너무 낮게 내려가 있지 않은지 확인하고, 바로 아래 **버튼을 사용하여 변경** 옵션도 같이 보세요. 이게 켜져 있으면 옆면 볼륨 버튼으로 알림 음량이 같이 바뀌어버릴 수 있습니다.

알람이 중요한 분이라면 이 옵션은 꺼두는 게 안전합니다.

## 주시 지각 기능도 의심해보세요

알람이 울리긴 했는데 소리가 너무 작아서 못 들은 것 같다면, 주시 지각 기능도 한 번 확인해보세요.

화면을 보고 있는지 감지해서 일부 소리나 화면 동작을 조절해주는 기능인데, 평소엔 편해도 알람처럼 무조건 크게 울려야 하는 상황에서는 오히려 방해가 될 수 있어요.

```
설정 > Face ID 및 암호 > 주시 지각 기능
```

꺼둔 뒤 알람 소리가 달라지는지 비교해보세요. 모든 경우의 원인은 아니지만, 분명 울린 것 같은데 못 들었다면 의심해볼 만합니다.

## 자기 전 블루투스 연결도 체크하세요

에어팟이나 블루투스 스피커를 연결해놓고 잠들었다면, 연결 상태도 한 번 확인해보세요. 알람이 진짜 안 울린 게 아니라, 소리가 다른 데서 나고 있었을 수도 있거든요.

중요한 날 전날엔 아예 블루투스를 꺼두고 자는 게 제일 속 편해요.

```
설정 > 블루투스 > 끔
```

또는 제어센터에서 오디오 출력이 어디로 잡혀 있는지 확인해보세요. 에어팟이나 스피커, 차량 오디오처럼 못 듣는 장치에 연결돼 있으면 알람을 놓친 것처럼 느껴질 수 있어요.

## 아이폰 업데이트 알람 안 울림 그래도 불안하면 며칠은 보조 알람도 같이 써두세요

이게 무조건 업데이트 버그다, 라고 단정하긴 어려워요. 알람 자체 문제일 수도 있고, 수면 집중 모드, 건강 앱 기상 알람, 음량, 주시 지각 기능, 블루투스 연결이 겹쳐서 생긴 문제일 수도 있고요.

다만 알람은 한 번만 놓쳐도 하루가 통째로 꼬이는 거라, 업데이트 후 한 번이라도 이상했다면 며칠 정도는 보조 알람을 같이 써두는 걸 추천해요. 애플워치든 다른 휴대폰이든 일반 알람시계든 뭐든 좋습니다. 아이폰 알람이 다시 안정적으로 울리는 게 확인될 때까지는요.

## 아이폰 업데이트 알람 안 울림 마치며

결국 저는 위에 정리한 것들 하나씩 다 확인해봤는데, 제 경우엔 수면 집중 모드랑 알람이 미묘하게 겹치는 게 문제였더라고요.

분명 시계 앱 알람은 정상적으로 켜져 있었는데, 수면 집중 모드 종료 시간이랑 알람 시간이 묘하게 안 맞아서 가끔 알람이 막히는 느낌이었던 거예요.

그래서 일단 단축어로 알람 1분 전에 수면 집중 모드를 자동으로 꺼버리게 해놨어요. 그 이후로는 10분 단위로 맞춰둔 알람이 다시 다 하나씩 제대로 울리기 시작했습니다.

정확히 무슨 원리로 꼬였던 건지까지는 저도 모르겠지만, 일단 지금은 안 늦고 잘 일어나고 있어요.

알람 때문에 이미 한두 번 식겁한 적 있다면, 설정 다 뒤져보기 전에 이 자동화부터 먼저 해보세요. 그래도 며칠은 좀 불안할 수 있으니, 애플워치나 다른 휴대폰 같은 보조 알람 하나 정도는 같이 켜두는 걸 추천합니다.

알람은 똑똑한 것보다 그냥 확실하게 울려주는 게 제일이니까요.
