## 크롬에 저장된 비밀번호 확인하는 방법

비밀번호가 기억 안 날 때 가장 먼저 확인할 곳은 크롬의 비밀번호 관리자입니다.

예전에 로그인할 때 “비밀번호를 저장하시겠습니까?”에서 저장을 눌렀다면, 크롬 안에 비밀번호가 남아 있을 수 있습니다.

PC에서는 크롬 주소창에 아래 주소를 입력하면 바로 이동할 수 있습니다.

```
chrome://password-manager
```

![크롬 비밀번호 관리자 로 접속합니다.](https://spacecatx.me/wp-content/uploads/2026/04/image-23-1024x440.png)

주소를 입력하면 크롬 비밀번호 관리자 화면이 열립니다.  
여기서 검색창에 비밀번호를 확인하고 싶은 사이트 이름을 입력하세요.

예를 들어 네이버 비밀번호가 기억 안 난다면 **naver**라고 검색하면 됩니다.

![사이트나 서비스명으로 검색합니다!](https://spacecatx.me/wp-content/uploads/2026/04/image-24-1024x489.png)

저장된 계정이 나오면 해당 항목을 클릭합니다.

![비밀번호 오른쪽의 눈 모양 아이콘을 클릭하면 비밀번호가 노출 됩니다.](https://spacecatx.me/wp-content/uploads/2026/04/image-25-1024x604.png)

그다음 비밀번호 옆에 있는 **눈 모양 아이콘**을 누르면 됩니다.

눈 모양 아이콘을 누르면 바로 비밀번호가 보이는 것은 아니고, 한 번 더 본인 인증을 거칩니다.

윈도우라면 PIN 번호나 윈도우 로그인 비밀번호를 입력해야 하고, 맥이라면 맥 로그인 비밀번호나 Touch ID 인증이 필요할 수 있습니다.

인증이 끝나면 숨겨져 있던 비밀번호가 표시됩니다.

> 비밀번호 옆 눈 모양 아이콘을 누르면 기기 인증 후 저장된 비밀번호를 확인할 수 있습니다.

[여기에 PC 크롬 비밀번호 관리자 캡처 삽입]

## 모바일 크롬에서도 확인할 수 있습니다

PC가 없더라도 모바일 크롬 앱에서 저장된 비밀번호를 확인할 수 있습니다.

아이폰 기준으로는 크롬 앱을 열고, 오른쪽 아래의 **점 세 개 메뉴**를 누릅니다.  
안드로이드에서는 기기에 따라 점 세 개 메뉴가 오른쪽 위에 있을 수 있습니다.

그다음 아래 순서로 들어가세요.

(1) 크롬 앱 실행  
(2) 점 세 개 메뉴 선택  
(3) 설정 선택  
(4) 비밀번호 관리자 선택  
(5) 검색창에 사이트 이름 입력  
(6) 저장된 계정 선택  
(7) 눈 모양 아이콘 선택  
(8) Face ID, 지문, 화면 잠금 인증 후 비밀번호 확인

모바일에서도 원리는 같습니다.

저장된 사이트를 찾고, 비밀번호 옆의 눈 모양 아이콘을 누른 뒤, 본인 인증을 하면 비밀번호를 확인할 수 있습니다.

## 저장된 비밀번호가 안 보일 때 확인할 것

비밀번호 관리자에 들어갔는데 원하는 사이트가 안 보일 수도 있습니다.  
이럴 때는 아래를 먼저 확인해보세요.

(1) 크롬에 비밀번호를 저장한 적이 있는지 확인하기  
(2) 다른 구글 계정으로 로그인되어 있지 않은지 확인하기  
(3) 예전에 쓰던 PC나 휴대폰에 저장되어 있지 않은지 확인하기  
(4) 사이트 이름을 다르게 검색해보기  
(5) 시크릿 모드에서 로그인했던 것은 아닌지 떠올려보기

특히 구글 계정을 여러 개 쓰는 경우에는, 크롬에 로그인된 계정이 달라서 비밀번호가 안 보일 수 있습니다.  
예전에 쓰던 계정으로 크롬에 로그인되어 있었는지도 확인해보세요.

사이트 이름도 꼭 정확히 검색할 필요는 없습니다.

네이버라면 **naver**, 구글이라면 **google**, 쿠팡이라면 **coupang**처럼 짧게 검색해보는 게 더 잘 나올 때가 있습니다.

생각보다 내 비밀번호는 내 기억보다 크롬이 더 잘 기억하고 있을 때가 있습니다!

비밀번호 재설정부터 눌렀으면 또 본인인증하고, 다른 기기 로그인도 다시 챙겨야 했을 텐데 말이죠.
