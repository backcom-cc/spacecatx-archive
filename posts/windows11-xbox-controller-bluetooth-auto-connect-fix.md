## 엑스박스 패드 블루투스 자동연결 안됨 증상 해결 방법

### 0. 먼저 블루투스 장치를 제거한 뒤 다시 페어링하세요

이건 먼저 해주셔야 합니다.  
그래야 엑스박스 패드 자동 연결 문제를 제대로 점검할 수 있습니다. 기존에 꼬인 등록 정보가 남아 있으면, 아래 방법들을 적용해도 증상이 헷갈릴 수 있습니다.

```
설정 → Bluetooth 및 장치 → 장치 → Xbox Wireless Controller 선택 → 제거
```

이후 패드 상단 페어링 버튼을 눌러 다시 연결하시면 됩니다.  
그래도 같은 문제가 반복된다면, 아래 방법들을 순서대로 확인해보세요.

---

### 1. 블루투스 전원 관리부터 끄세요

윈도우 11은 배터리를 아끼려고 블루투스 장치를 알아서 잠재울 때가 있습니다.  
그러면 엑스박스 패드가 등록은 되어 있어도, 다시 켰을 때 자동 연결이 안 붙을 수 있습니다. 그래서 먼저 이 절전 기능부터 꺼두는 게 좋습니다.

```
장치 관리자 → Bluetooth → 사용 중인 블루투스 어댑터 우클릭 → 속성 → 전원 관리 → 
전원을 절약하기 위해 컴퓨터가 이 장치를 끌 수 있음 체크 해제
```

---

### 2. Bluetooth Support Service를 다시 시작하세요

블루투스 자체가 꼬이면, 패드는 저장돼 있는데 실제 연결만 안 붙는 일이 생길 수 있습니다.  
이럴 때는 서비스부터 다시 시작해서 블루투스 기능을 한 번 새로 깨워보는 게 좋습니다.

```
Win + R → services.msc → Bluetooth Support Service → 다시 시작
```

---

### 3. 블루투스 드라이버를 업데이트하세요

전원 관리까지 껐는데도 자동 연결이 계속 안 되면, 이제는 블루투스 드라이버를 의심해볼 차례입니다.  
윈도우 쪽 드라이버가 오래됐거나 꼬여 있으면, 엑스박스 패드가 매번 다시 등록돼야 하는 상황이 생길 수 있습니다.

```
장치 관리자 → Bluetooth → 사용 중인 블루투스 어댑터 우클릭 → 드라이버 업데이트
```

---

### 4. 엑스박스 패드 펌웨어도 확인하세요

패드를 오래 안 썼다면, 패드 쪽 소프트웨어가 예전 상태일 수도 있습니다.  
이 경우 PC 쪽 블루투스는 멀쩡한데, 패드가 자동 연결을 매끄럽게 못 잡는 경우가 있습니다.

```
Microsoft Store → Xbox Accessories 설치 → 패드 연결 → 업데이트 확인
```

공식 안내가 필요하시면 아래 Xbox 지원 문서도 같이 참고해보세요.

[Xbox 무선 컨트롤러 업데이트 공식 문서 보기](https://support.xbox.com/en-US/help/hardware-network/controller/update-xbox-wireless-controller)

---

### 5. USB 포트 위치나 무선 간섭도 확인해보세요

데스크탑에서는 이것도 은근 자주 걸립니다.  
블루투스 동글이 본체 뒤쪽 USB 포트에 꽂혀 있거나, 주변에 USB 3.0 장치가 많으면 신호가 약해져 자동 연결이 불안정해질 수 있습니다.

특히 엑스박스 패드는 연결은 되는데 끊기거나, 켰을 때 바로 안 붙는 형태로 나타나는 경우도 있습니다.

이 부분은 환경마다 경우의 수가 많아서, [블루투스 끊김 해결 글](https://spacecatx.me/xbox-controller-bluetooth-disconnect-windows11-fix/)에서 더 자세하게 정리해두었습니다.  
무선 간섭, USB 3.0 간섭, 동글 위치 변경, 공유기 2.4GHz 영향까지 궁금하시면 아래 글도 같이 참고해보세요.

### 6. 그래도 안 되면 블루투스를 껐다 켜고, 필요하면 어댑터를 재설치하세요

여기까지 했는데도 자동 연결이 안 되면, 블루투스 기능 자체를 한 번 껐다 켜고, 필요하면 어댑터를 삭제 후 재설치하는 쪽으로 가야 합니다.

```
설정 → Bluetooth 및 장치 → Bluetooth 끄기 → 몇 초 후 다시 켜기
```

그래도 안 되면 아래 단계까지 진행해보세요.

```
장치 관리자 → Bluetooth → 블루투스 어댑터 우클릭 → 장치 제거 → 재부팅
```

## 엑스박스 패드 블루투스 자동연결 안됨 왜 이런 일이 생길까?

이런 증상은 보통 블루투스 등록 정보가 꼬였거나, 윈도우 11의 전원 관리 때문에 블루투스 장치가 잠들면서 생기는 경우가 많습니다.  
여기에 드라이버 문제나 무선 간섭까지 겹치면, **페어링은 되어 있는데 실제로는 자동 연결이 안 되는 상황**이 생길 수 있습니다.
