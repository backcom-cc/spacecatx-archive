# 유튜브 앱 검색 오류? 검색버튼 누르면 예전 검색어 반복됨 해결 방법

##### 유튜브 앱 검색 오류 지워도 다시 뜰 때, 먼저 확인할 것

유튜브 앱에서 검색어를 지웠는데도 이전 검색어가 다시 뜰 때가 있습니다.  
새로 검색하려고 검색창을 눌렀는데 예전에 입력했던 단어가 먼저 보이면, 앱이 고장 난 것처럼 느껴져요.

하지만 이 증상은 무조건 앱 오류가 아닙니다.  
유튜브 검색 기록, 최근 검색어 표시, 키보드 자동완성이 서로 다르게 보이는 경우가 많습니다.

그래서 처음부터 앱을 삭제하거나 캐시를 지울 필요는 없습니다.  
먼저 이전 검색어가 어디에 다시 뜨는지 확인하고, 검색 기록부터 정리하세요.

---

## 먼저 검색어가 어디에 뜨는지 확인하세요

유튜브 검색어가 다시 뜬다고 해서 전부 같은 문제는 아닙니다.  
이전 검색어가 보이는 위치에 따라 해결 방법이 달라집니다.

* 검색창 아래 목록에 뜬다 = 유튜브 검색 기록을 확인하세요.
* 검색창 안에 자동으로 입력된다 = 앱을 완전히 종료한 뒤 다시 실행하세요.
* 키보드 위 추천 단어로 뜬다 = 키보드 자동완성을 확인하세요.
* 다른 기기에서도 똑같이 뜬다 = 구글 계정의 YouTube 기록을 확인하세요.

**검색어가 다시 뜬다고 해서 무조건 앱을 지울 필요는 없습니다.**  
검색어가 어디에서 다시 보이는지부터 확인하세요.

---

## 검색창 아래에 뜨면 검색 기록부터 삭제하세요

유튜브 검색창을 눌렀을 때 아래쪽에 이전 검색어 목록이 뜬다면, 먼저 검색 기록을 삭제하세요.

유튜브 앱에서는 검색 기록을 개별로 지울 수도 있고, 일정 기간의 기록을 삭제할 수도 있습니다.  
검색 기록을 더 이상 저장하고 싶지 않다면 YouTube 기록 설정에서 저장을 끌 수도 있어요.

유튜브 앱에서는 아래 경로로 들어가면 됩니다.

```
프로필 사진 > 설정 > 전체 기록 관리 > 삭제 또는 제어
```

유튜브 검색 기록 삭제와 일시중지는 [YouTube 고객센터의 검색 기록 관리 안내](https://support.google.com/youtube/answer/57711)에서도 확인할 수 있습니다.

---

## 검색어를 지웠는데 다시 보이면 계정 기록을 확인하세요

검색창에서 검색어를 지웠는데도 다시 보인다면, 앱 화면에서만 지운 상태일 수 있습니다.  
이럴 때는 구글 계정에 저장된 YouTube 기록까지 확인하세요.

스마트폰, 태블릿, PC에서 같은 구글 계정으로 유튜브를 쓰고 있다면 기록이 여러 기기에서 이어집니다.

이럴 때는 유튜브 앱 안에서만 보지 말고, 전체 기록 관리 화면에서 삭제 상태를 확인하세요.

* 특정 검색어만 삭제했는지
* 전체 검색 기록을 삭제했는지
* YouTube 기록 저장이 켜져 있는지
* 다른 기기에서도 같은 검색어가 보이는지

다른 기기에서도 같은 검색어가 계속 보이면, 앱 하나의 문제가 아닙니다.  
계정 기록부터 정리하세요.

---

## 키보드 위에 뜨면 유튜브 문제가 아닙니다

이전 검색어가 유튜브 검색창 아래가 아니라 키보드 위에 추천 단어처럼 뜬다면, 유튜브 검색 기록이 아닙니다.

이 경우에는 스마트폰 키보드가 예전에 입력한 단어를 자동완성으로 보여주는 것입니다.

예를 들면 이런 경우입니다.

* 검색창 아래 목록이 아니라 키보드 바로 위에 단어가 뜸
* 유튜브뿐 아니라 다른 앱 검색창에서도 비슷한 단어가 뜸
* 검색 기록을 지웠는데도 키보드 추천 단어로 남아 있음

이럴 때는 유튜브 앱을 지워도 해결되지 않습니다.  
키보드 설정에서 자동완성, 추천 문구, 학습 데이터 삭제 항목을 확인하세요.

갤럭시는 삼성 키보드나 Gboard 설정을 확인하세요.  
아이폰은 키보드 사전 재설정이나 자동완성 설정을 확인하면 됩니다.

---

## 검색창 안에 자동으로 입력되면 앱을 다시 실행하세요

검색창 아래 목록이 아니라 검색창 안에 이전 검색어가 자동으로 들어가 있는 것처럼 보일 때도 있습니다.

이 경우에는 앱 화면이 이전 상태를 계속 보여주는 상황일 수 있습니다.  
먼저 유튜브 앱을 완전히 종료한 뒤 다시 실행하세요.

아이폰은 앱 전환 화면에서 유튜브 앱을 위로 밀어 종료하세요.  
갤럭시는 최근 앱 화면에서 유튜브 앱을 닫으면 됩니다.

그다음 유튜브 앱을 다시 열고 검색창을 눌러보세요.  
단순한 화면 오류라면 이 단계에서 정리됩니다.

---

## 다른 계정이나 시크릿 모드로 비교하세요

검색 기록을 지웠는데도 같은 검색어가 계속 보이면, 계정 문제인지 앱 문제인지 나눠서 확인하세요.

가장 쉬운 방법은 다른 계정이나 시크릿 모드로 비교하는 것입니다.

* 다른 구글 계정으로 바꿨을 때 검색어가 안 뜬다 = 기존 계정 기록 문제입니다.
* 시크릿 모드에서 검색어가 안 뜬다 = 기존 계정 기록이나 추천 기록 문제입니다.
* 계정을 바꿔도 똑같이 검색창이 이상하다 = 앱이나 기기 쪽 문제입니다.

이렇게 확인하면 무작정 앱을 삭제하기 전에 문제 범위를 좁힐 수 있습니다.

---

## 그래도 안 되면 캐시 삭제나 재설치를 마지막에 하세요

여기까지 확인했는데도 검색창이 먹통처럼 보이거나, 키보드가 올라오지 않거나, 이전 검색어가 계속 자동 입력된다면 앱 쪽 조치를 하세요.

갤럭시나 안드로이드에서는 YouTube 앱 캐시를 삭제하세요.

```
설정 > 애플리케이션 또는 앱 > YouTube > 저장공간 > 캐시 삭제
```

아이폰은 YouTube 앱 캐시만 따로 지우는 메뉴가 없습니다.  
앱을 삭제한 뒤 다시 설치하는 방식이 가장 빠릅니다.

다만 이 방법은 마지막에 해도 됩니다.  
대부분은 검색 기록, 계정 기록, 키보드 자동완성 쪽을 확인하는 것만으로 원인을 구분할 수 있습니다.

---

## 정리하면

유튜브 앱에서 이전 검색어가 계속 다시 뜬다면, 먼저 앱 오류라고 단정하지 마세요.

검색어가 어디에 보이는지부터 확인해야 합니다.

1. 검색창 아래에 뜨면 유튜브 검색 기록을 삭제하세요.
2. 지워도 다시 보이면 구글 계정의 YouTube 기록 관리를 확인하세요.
3. 키보드 위에 뜨면 키보드 자동완성을 확인하세요.
4. 검색창 안에 자동 입력되면 앱을 완전히 종료한 뒤 다시 실행하세요.
5. 그래도 안 되면 마지막에 캐시 삭제나 앱 재설치를 하세요.

**유튜브 검색어가 다시 뜬다고 해서 무조건 앱을 지울 필요는 없습니다.**  
검색 기록인지, 계정 기록인지, 키보드 자동완성인지 먼저 구분하세요!
