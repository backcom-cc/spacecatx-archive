#### HDMI 연결 소리 안남? HDMI로 연결했는데 화면은 나오고 소리만 안 난다면, 먼저 윈도우 출력 장치를 확인하세요.

이 문제는 모니터 고장보다 출력 장치가 바뀌었거나, HDMI 오디오 장치가 제대로 잡히지 않았을 때 더 자주 발생합니다. 아래 상황 중 하나에 해당한다면, 표시한 순서부터 확인해보세요.

| 상황 | 먼저 의심할 원인 |
| --- | --- |
| 모니터를 바꾼 뒤 갑자기 소리가 안 난다 | 출력 장치가 새 모니터로 바뀐 경우 → 1번 |
| 노트북을 TV에 연결했는데 소리만 안 나온다 | TV가 출력 장치로 잡히지 않았거나 HDMI 오디오가 꼬인 경우 → 1번, 3번 |
| 사운드바나 외부 스피커를 쓰고 있다 | 기존 스피커 대신 모니터가 선택된 경우 → 1번 |
| 듀얼 모니터를 연결한 뒤 소리가 사라졌다 | 소리가 다른 모니터나 HDMI 장치로 나가는 경우 → 1번 |
| DP에서 HDMI로 바꿔 꽂은 뒤 소리가 안 난다 | HDMI 오디오 장치가 새로 잡히지 않은 경우 → 2번, 3번 |
| 그래픽 드라이버 업데이트 후 HDMI 소리만 안 나온다 | HDMI 오디오 드라이버 문제 가능성 → 3번 |
| 모니터에 스피커가 없다 | 출력 장치가 스피커 없는 모니터로 잡혔는지 확인 필요 → 1번 |

### HDMI 연결했는데 소리가 안 나는 이유

이 증상은 보통 아래 다섯 가지 안에서 해결되는 경우가 많습니다.

* 출력 장치 설정 문제
* 소리 제어판에서 장치가 비활성화된 경우
* 그래픽카드의 HDMI 오디오 드라이버 문제
* 모니터 자체 음소거 또는 볼륨 문제
* HDMI 케이블 연결 불량

가장 흔한 원인은 역시 **출력 장치가 엉뚱하게 바뀌는 경우**입니다.

---

### 1. 출력 장치 기본값 확인

윈도우가 가끔 기존 스피커나 사운드바 대신, 새로 연결한 모니터를 기본 출력 장치로 잡아버릴 때가 있습니다. 이 경우 화면은 잘 나오는데 소리만 다른 곳으로 나가게 됩니다.

확인 방법은 아래와 같습니다.

1. 작업표시줄 오른쪽 아래 **스피커 아이콘**을 우클릭합니다.
2. **소리 설정**으로 들어갑니다.
3. **출력 장치 선택**에서 현재 사용 중인 스피커 또는 사운드바가 선택되어 있는지 확인합니다.

새 모니터 이름이 잡혀 있거나, 평소 쓰지 않던 장치가 선택돼 있다면 그게 원인일 가능성이 큽니다.

---

### 2. 소리 제어판에서 사용 안 함 해제

설정 화면에서는 장치가 보이는데 실제로 소리가 안 날 때는, 제어판 쪽에서 장치가 비활성화돼 있는 경우도 있습니다.

이럴 때는 아래처럼 확인해 보면 됩니다.

1. `Win + R`을 누릅니다.
2. `mmsys.cpl`을 입력하고 실행합니다.
3. **재생** 탭으로 들어갑니다.
4. 빈 공간을 우클릭한 뒤 **사용할 수 없는 장치 표시**를 체크합니다.

회색으로 비활성화된 장치가 보이면 우클릭해서 **사용**으로 바꿔 주세요.

---

### 3. 그래픽카드 HDMI 오디오 드라이버 재설치

HDMI 소리는 단순히 케이블만의 문제가 아니라, 그래픽카드의 오디오 드라이버가 함께 관여합니다. 그래서 드라이버 업데이트 이후 HDMI 소리만 꼬이는 경우도 있습니다.

확인 순서는 아래와 같습니다.

1. **장치 관리자**를 엽니다.
2. **사운드, 비디오 및 게임 컨트롤러** 항목으로 들어갑니다.
3. `NVIDIA High Definition Audio` 또는 `AMD High Definition Audio`를 찾습니다.
4. 우클릭 후 **디바이스 제거**를 선택합니다.
5. 그다음 상단 메뉴에서 **하드웨어 변경 사항 검색**을 눌러 다시 잡히는지 확인합니다.

HDMI 오디오가 다시 정상 등록되면서 소리가 돌아오는 경우가 있습니다.

윈도우 오디오 문제 전반은 Microsoft 공식 도움말의 [Windows 소리 또는 오디오 문제 해결 안내](https://support.microsoft.com/en-us/windows/fix-sound-or-audio-problems-in-windows-73025246-b61c-40fb-671a-2535c7cd56c8)도 함께 참고할 수 있습니다.

드라이버 업데이트나 롤백이 필요하다면 Microsoft 공식 도움말의 [장치 관리자에서 드라이버 업데이트하는 방법](https://support.microsoft.com/en-us/windows/update-drivers-through-device-manager-in-windows-ec62f46c-ff14-c91d-eead-d7126dc1f7b6)을 참고하세요.

---

### 4. 모니터 자체 볼륨 또는 음소거 확인

만약 모니터에 스피커가 달린 모델이라면, PC 설정이 아니라 모니터 자체 설정 문제일 수도 있습니다. 특히 출력이 모니터로 바뀐 상태인데 모니터 볼륨이 0이거나 음소거돼 있으면 당연히 소리가 안 납니다.

모니터 하단이나 뒷면 버튼으로 OSD 메뉴(모니터에 달린 버튼으로 여는 메뉴)를 열고, 아래 항목을 확인해 보세요.

* Audio
* Volume
* Mute

생각보다 이 설정이 원인인 경우가 많습니다!

---

### 5. HDMI 케이블 다시 꽂기

HDMI는 영상과 음성을 함께 보내는 규격이라, 케이블이 애매하게 연결되면 화면은 잘 나오는데 소리만 이상하게 안 잡히는 경우가 있습니다.

> 영상은 나오는데 소리만 안 날 때는, 케이블을 완전히 뽑았다가 다시 꽂는 것만으로 해결되는 경우도 생각보다 많습니다.

본체와 모니터 양쪽 HDMI 케이블을 완전히 분리한 뒤 5초 정도 기다렸다가, 다시 끝까지 확실하게 꽂아 보세요.

---

### 정리

저는 모니터를 4K로 교체한 직후 이 문제를 겪었는데, 원인은 1번이었습니다. 사운드바를 따로 쓰고 있었는데 PC가 출력 장치를 사운드바 대신 새 모니터로 바꿔버린 상태였고, 소리는 스피커도 없는 모니터 쪽으로 나가고 있었던 거죠. 출력 장치만 다시 잡아주니 바로 해결됐습니다.

**HDMI 연결 후 소리가 안 난다면 고장보다 설정을 먼저 의심하세요.** 대부분은 출력 장치가 바뀌었거나 HDMI 오디오가 제대로 잡히지 않은 게 원인입니다.
