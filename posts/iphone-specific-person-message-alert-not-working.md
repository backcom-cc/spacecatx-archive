## 이상하게 아이폰 특정 사람 문자 알림 안옴 아이폰 문자는 오는데 알림이 안 울렸어요

회사 아이폰을 쓰기 시작하면서 거래처 분들과 문자로 업무를 볼 일이 생겼어요.

여러 분들께 확인 요청을 드렸고, 답변을 취합해야 했어요. 근데 유독 한 분만 답장이 안 오더라고요. 혹시 못 받으신 건지, 바쁘신 건지, 계속 신경 쓰이면서 일했어요. 퇴근 시간이 가까워져도 감감무소식이었고요.

어쩔 수 없이 다른 분들 답변부터 취합하려고 문자함을 정리하다가, 그 사이에서 찾아냈어요.

다섯 개나 되는 그분의 문자 더미를요.

엥. 뭐지. 알림은 단 한 번도 울리지 않았는데, 언제 온 거지? 하루 종일 기다렸던 게 허탈하기도 하고, 뭔가 이상하다 싶었어요.

이 글에서는 아이폰에서 문자는 오는데 알림이 안 울리는 원인이 뭔지, 특히 특정 사람한테서만 그런 경우 어디를 확인해야 하는지 정리해드릴게요.

> **아이폰에서 특정 사람 문자 알림만 안 온다면, 문자 수신 문제보다 해당 대화방의 `알림 가리기`가 켜져 있는지 먼저 확인해보세요.**

---

## 먼저 해당 대화방의 알림 가리기를 확인하세요

이럴 때 가장 먼저 볼 곳은 메시지 앱 안에 있는 **알림 가리기** 설정이에요.

아이폰은 전체 문자 알림을 끄지 않아도, 특정 사람 대화방만 따로 조용하게 만들 수 있어요. 카카오톡 알림 끄기랑 똑같은 기능이에요. 이게 켜져 있으면 문자는 정상적으로 도착해요. 메시지 앱 열면 대화도 그대로 있고요.

문제는 알림이에요.

소리도 안 울리고, 진동도 없고, 잠금화면이나 배너도 안 떠요. 그러니까 상대방은 분명히 답장을 보냈는데 저는 하루 종일 “왜 답이 없지?” 하고 기다리는 상황이 생기는 거예요.

```
메시지 앱 > 해당 대화방 > 상단 이름 또는 프로필 아이콘 > 알림 가리기 확인
```

켜져 있다면 꺼주세요. 문자가 안 온 게 아니라, 아이폰이 그 대화방만 조용히 처리하고 있었던 거예요.

메시지 목록에서 대화방을 왼쪽으로 밀다가 실수로 알림을 끄는 경우도 많아요. 건드린 기억이 없어도 한 번은 꼭 확인해보세요.

애플 공식 안내에서도 특정 대화방의 알림을 끄거나 다시 켜는 방법을 설명하고 있으니, iOS 버전 떄문에 메뉴명이 헷갈린다면 함께 확인해보세요.

[Apple 공식 안내: iPhone에서 메시지 알림이 오지 않을 때 확인하기](https://support.apple.com/en-us/108127)

## 연락처별 문자음이 없음으로 되어 있는지 확인하세요

`알림 가리기`가 꺼져 있는데도 이상하다면, 이번엔 그 사람에게 따로 설정된 **문자음**을 확인해보세요.

아이폰은 연락처마다 문자 수신음을 다르게 설정할 수 있어요. 반대로 말하면, 특정 사람의 문자음만 없음으로 바뀌어 있을 수도 있다는 거예요.

이 경우엔 문자는 오고, 알림도 화면에 뜰 수 있어요. 근데 소리가 안 나니까 지나치게 되는 거예요.

```
메시지 앱 > 해당 대화방 > 상단 이름 또는 프로필 아이콘 > 정보 > 편집 > 문자음 확인
```

`없음`으로 되어 있다면 다른 알림음으로 바꿔주세요.

잠금화면이나 알림센터에는 문자가 보이는데 소리만 안 났다면, 알림 가리기보다 이쪽을 먼저 봐야 해요.

---

## 집중 모드에서 해당 연락처 알림이 막혀 있는지 확인하세요

앞의 두 가지가 다 정상이라면, 이번엔 **집중 모드**를 확인해보세요.

집중 모드는 방해금지, 수면, 업무 모드처럼 상황에 맞게 알림을 조용히 만드는 기능이에요. 이 설정에 따라 특정 사람 알림만 허용되거나, 반대로 일부 사람 알림이 조용히 처리될 수 있어요.

```
설정 > 집중 모드 > 현재 사용 중인 모드 > 사람 > 알림 허용 또는 무음 설정 확인
```

해당 사람이 허용 목록에서 빠져 있거나 무음 처리로 설정돼 있다면 알림이 안 울려요.

특히 항상 안 오는 게 아니라 업무 시간이나 밤 시간대에만 조용하다면 집중 모드 쪽을 꼭 확인해보세요. 저처럼 회사 아이폰을 쓰는 경우, 켠 기억이 없어도 시간이나 위치 조건에 따라 자동으로 켜져 있을 수 있어요.

애플 공식 안내에서도 특정 대화방의 알림을 끄거나 다시 켜는 방법을 설명하고 있으니, 메뉴명이 헷갈린다면 함께 확인해보세요.

[Apple 공식 안내: iPhone에서 메시지 알림이 오지 않을 때 확인하기](https://support.apple.com/en-us/108127)

## 메시지 앱 전체 알림 설정도 한 번 확인해보세요

앞의 설정이 다 정상이라면 마지막으로 메시지 앱 알림 설정 자체를 봐야 해요.

특정 사람만 문제인 줄 알았는데, 알고 보면 메시지 앱 전체 알림이 꺼져 있거나 배너, 잠금화면 알림이 빠져 있는 경우도 있거든요.

```
설정 > 알림 > 메시지 > 알림 허용 확인
```

`알림 허용`이 꺼져 있다면 켜주세요. 그리고 아래 항목도 같이 봐요.

```
설정 > 알림 > 메시지 > 잠금 화면 / 알림 센터 / 배너 체크
```

셋 중 일부가 꺼져 있으면 알림이 온 것 같지 않게 느껴질 수 있어요. 소리는 났는데 화면에 안 뜨거나, 잠금화면에선 안 보이고 알림센터에만 남는 식으로요.

사운드 설정도 같이 확인해보세요.

```
설정 > 알림 > 메시지 > 사운드
```

`없음`으로 되어 있으면 문자 알림이 와도 소리가 안 나요.

## 아이폰 특정 사람 문자 알림 안옴, 그래도 안 된다면?

위 설정을 다 확인했는데도 계속 이상하다면 아이폰을 한 번 재시동해보세요. 그래도 반복된다면 iOS 업데이트 여부를 확인하고, 메시지 앱을 완전히 종료했다가 다시 열어보는 것도 방법이에요.

한 가지만 짚고 넘어갈게요.

이 글은 **문자는 오는데 알림만 안 울리는 경우**를 기준으로 썼어요. 문자 자체가 아예 안 온다거나 메시지 앱에서도 안 보인다면 얘기가 달라요. 그 경우엔 차단된 연락처, 알 수 없는 발신자 필터링, iMessage 설정, 통신사 수신 문제 쪽을 따로 확인해야 해요.

메시지가 도착해 있는데 알림만 조용했다면 `알림 가리기`부터, 문자 자체가 안 보인다면 수신 문제 쪽으로 봐야 한다는 거 기억해두세요.

## 아이폰 특정 사람 문자 알림 안옴 마치며

저는 그날 이후로 새 연락처랑 문자 시작하면 알림 가리기 켜져 있는지 한 번씩 확인하는 습관이 생겼어요. 그분 연락처를 일부러 건드린 기억은 없는데, 아마 문자가 몇 번 오고가다가 어쩌다 눌렀나봐요. 건드린 기억이 없어도 켜져 있는 경우가 생각보다 많더라고요.

특정 사람 문자만 조용하다면 수리나 업데이트 전에 이 글 순서대로 먼저 확인해보세요. 대부분은 설정 하나 바꾸는 걸로 해결돼요.
