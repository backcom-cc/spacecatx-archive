## 아이폰 업데이트 후 오타 많아짐, 키보드도 느려졌어요.

저는 회사 업무를 아이폰으로 많이 봐요. 메일, 메시지, 카카오톡까지. 오타가 나면 좀 곤란한 상황이 많아서 자동 수정도 켜두고, 나름 조심히 쓰는 편이었거든요.

그런데 어느 날부터 타이핑이 좀 이상한 거예요. 분명 제가 누른 글자가 아닌 옆 글자가 입력되는 느낌? 키보드가 살짝 끈적거리는 것처럼 반응이 늦고, 제가 치는 속도를 못 따라오는 것 같더라고요.

처음엔 제가 실수했겠거니 했어요. 워낙 자판도 작고 하니까. 근데 유난히 업데이트 이후부터 그런 느낌이 강해진 것 같더라고요.

자동 수정도 예전 같지 않더라고요. 분명 맞는 단어인데 이쌍한걸로 고쳐버리거나, 고쳐줬으면 하는 건 그냥 냅두거나. 업무 메시지 보내기 전에 몇 번씩 다시 확인해야 하는 상황이 돼버렸어요ㅠ..

이 글에서는 업데이트 후 키뾰ㄷ 반응ㅇㅣ 느려지거나 오타가 자ㅈ아졌을 때 확인할 설쪙드를 정리해드릴게요. 아이폰아 살ㄹㅕ줘ㅓ 제댜로 하란말이야 ㅠ

> **한줄결론:** 아이폰 업데이트 후 오타가 많아졌다면, 슬라이드하여 입력과 자동 수정 설정부터 확인해보세요.

---

## 업데이트가 끝났는데도 이상하다면 재시동부터 해보세요

업데이트 직후에는 설치가 다 된 것처럼 보여도 앱이나 시스템이 뒤에서 정리되는 과정에서 키보드 반응이 평소랑 다르게 느껴질 수 있어요. 설정 건드리기 전에 일단 재시동 한 번 해보세요.

```
볼륨 올리기 짧게 누르기 > 볼륨 내리기 짧게 누르기 > 측면 버튼 길게 누르기
```

재시동 후에도 계속 이상하면 그때 아래 설정들을 하나씩 확인하면 돼요.

---

## 옆 글자가 자꾸 눌리는 느낌이면 슬라이드하여 입력을 꺼보세요

슬라이드하여 입력은 손가락을 떼지 않고 글자 위를 쓸어서 단어를 입력하는 기능인데요. 평소에 이 기능을 쓰지 않더라도 켜져 있으면, 빠르게 타이핑하다가 의도하지 않은 움직임이 입력으로 잡히는 경우가 있어요.

분명 한 글자씩 누르는데 옆 글자가 들어가거나 키보드가 너무 예민하게 반응하는 느낌이라면 이 설정부터 꺼보세요.

```
설정 > 일반 > 키보드 > 슬라이드하여 입력 > 끔
```

---

## 자동 수정이 오히려 방해된다면 잠깐 꺼보세요

저도 업무 메시지 보내다가 이런 걸 겪었어요.

“드릴게요” -> “들릴게요”  
“검토 부탁드립니다” -> “검도 부탁드립니다”  
“확인 부탁드립니다” -> “확인 부탁드린다”

협력사한테 갑자기 반존대하는 사람 됨. ㅠㅠ

친구 카톡이면 웃고 넘기겠는데 업무 메시지에서는 진짜 곤란하더라고요. 이럴 때는 자동 수정 잠깐 꺼두고 차이가 있는지 확인해보세요.

```
설정 > 일반 > 키보드 > 자동 수정 > 끔
```

키보드 위 추천 단어까지 엉뚱하게 느껴진다면 예측 텍스트도 같이 꺼보세요.

아이폰 키보드의 자동 수정과 예측 텍스트 설정은 애플 공식 안내에서도 확인할 수 있습니다.  
[Apple 공식 지원: 자동 수정 및 예측 텍스트 사용하기](https://support.apple.com/en-us/104995)

```
설정 > 일반 > 키보드 > 예측 텍스트 > 끔
```

---

## 계속 이상한 단어로 바뀐다면 키보드 사전을 재설정하세요

자동 수정 꺼봐도 비슷한 문제가 계속된다면 키보드 사전이 꼬였을 수 있어요. 예전에 잘못 입력한 단어를 아이폰이 기억하고 있는 경우가 있거든요. 다만 그동안 학습된 개인 단어 기록이 초기화되니까 앞의 설정들 먼저 해보고 그래도 안 되면 시도해보세요.

```
설정 > 일반 > 전송 또는 iPhone 재설정 > 재설정 > 키보드 사전 재설정
```

---

## 특정 키보드만 이상하다면 삭제 후 다시 추가해보세요

한국어 키보드에서만 유독 오타가 많거나 한쪽만 이상한 느낌이라면 키보드를 삭제하고 다시 추가해보는 방법도 있어요.

```
설정 > 일반 > 키보드 > 키보드
```

사용 중인 키보드를 왼쪽으로 밀어서 삭제한 뒤, 새로운 키보드 추가에서 다시 선택하면 돼요.

---

## 키보드 반응이 느리다면 저장 공간도 확인해보세요

오타라기보다 누른 뒤 한 박자 늦게 입력되는 느낌이라면 저장 공간도 확인해보세요. 업데이트 후에는 임시 데이터 때문에 저장 공간이 평소보다 더 줄어있을 수 있어요. 공간이 너무 부족하면 키보드뿐 아니라 전반적인 반응이 다 느려질 수 있거든요.

```
설정 > 일반 > iPhone 저장 공간
```

안 쓰는 앱이나 큰 파일 조금 정리해보시고, 추가 업데이트가 올라와 있는지도 같이 확인해보세요.

```
설정 > 일반 > 소프트웨어 업데이트
```

## 마치며

저는 슬라이드하여 입력 끄고, 자동 수정도 잠깐 꺼뒀다가 키보드 사전 재설정까지 해봤는데 조금 나아진 느낌이었어요. 극적으로 달라진 건 아니고, 그냥 좀 덜 거슬리는 정도?

그러다 iOS 27 베타를 설치하고 나서는 체감이 좀 더 나아진 것 같더라고요. 키보드 반응이 확실히 좀 더 빨라진 느낌이랄까요. 아직 정식 버전은 아니지만, 정식 업데이트 기다리고 계신 분들은 조금만 더 기다려보셔도 좋을 것 같아요.

일단 그 전까지는 위에 정리한 설정들 하나씩 해보시고, 저처럼 조금이라도 나아지길 바랍니다. 아이폰아 제발 ㅠ
