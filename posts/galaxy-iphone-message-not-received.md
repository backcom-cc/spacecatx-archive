## 갤럭시에서 아이폰으로 바꿨는데 문자 안옴? RCS부터 확인해야 해요!

저희 언니가 얼마 전에 갤럭시에서 아이폰으로 갈아탔어요.

오래 쓰던 갤럭시가 딱히 불만스러웠던 건 아닌데, 주변에서 다들 아이폰 쓰고 카메라 갬성도 탐나서 질렀다고 하더라고요. 그런데 바꾸고 나서 문자가 잘 안 온다는 거예요. 분명히 아이폰으로 바꿨는데 문자는 예전 갤럭시로 온다고요.

저도 처음에는 뭔 헛소린가, 언니가 유심 잘못 넣은 거 아냐? 막 쑤셔넣은 거 아냐? 했는데, 알고 보니 갤럭시에서 아이폰으로 넘어올 때 꽤 자주 생기는 문제더라고요.

이 글에서는 갤럭시에서 아이폰으로 바꾼 뒤 문자가 이상할 때 확인할 것들을 상황별로 정리해봤어요. 본인 상황에 맞는 것만 골라서 보시면 돼요.

> 한줄결론:  
> **갤럭시에서 아이폰으로 바꿨는데 문자가 안 온다면,**   
> **먼저 문자가 이전 갤럭시로 가는지 확인하고 RCS·채팅+ 기능부터 꺼보세요.**

## 혹시 문자가 이전 갤럭시폰으로 오나요? 그럼 RCS 기능부터 꺼보세요

갤럭시에서 아이폰으로 바꿨는데 새 아이폰이 아니라 이전 갤럭시폰으로 문자가 온다면, 예전 갤럭시에 남아 있는 RCS 기능부터 확인해보세요.

RCS가 뭔지 모르셔도 괜찮아요. 쉽게 말하면 **안드로이드판 아이메시지** 같은 거예요. 갤럭시끼리 문자를 주고받을 때 일반 SMS가 아니라 채팅형으로 처리되는 기능인데, 이게 예전 갤럭시에 아직 내 번호로 연결돼 있으면 상대방 갤럭시가 나를 아직 갤럭시 사용자로 인식해버려요.

그래서 문자가 새 아이폰이 아니라 예전 갤럭시로 가는 거고요.

예전 갤럭시폰이 아직 있다면 메시지 앱에서 RCS, 또는 채팅+ 기능을 꺼주면 돼요.

*설정*

![갤럭시에서 아이폰으로 바꿨는데 문자 안옴-갤럭시 설정3](https://spacecatx.me/wp-content/uploads/2026/06/image-4-1024x924.png)

*채팅+ 설정*

**삼성 기본 메시지 앱**을 쓰셨다면 아래 경로로 들어가보세요.

```
삼성 메시지 앱 > 더보기(⋮) > 설정 > 채팅+ 설정 > 채팅+ 관리 > 채팅+ 끄기
```

**Google 메시지 앱**을 쓰셨다면 아래 경로로 들어가면 됩니다.

```
Google 메시지 앱 > 프로필 아이콘 > 메시지 설정 > RCS 채팅 > RCS 채팅 사용 끄기
```

기종이나 One UI 버전, 통신사에 따라 메뉴 이름이 조금씩 다를 수 있어요. RCS 채팅, 채팅 기능, 채팅+, 고급 메시지 이런 식으로 보일 수 있는데, 핵심은 같습니다. **채팅형 문자 기능을 꺼주면 된다**고 생각하시면 돼요.

## 예전 갤럭시폰이 없으면 Google RCS 비활성화 페이지를 이용하세요

이미 예전 갤럭시폰을 초기화했거나 중고로 팔아버렸다면 메시지 앱에서 직접 끄는 건 당연히 안 되죠. 이럴 때는 Google에서 제공하는 **RCS 비활성화 페이지**를 쓰면 돼요. 폰이 없어도 번호만 있으면 진행할 수 있습니다.

구글에서 **Google RCS 비활성화**로 검색하거나 아래 주소로 바로 들어가세요.

[Google RCS 채팅 비활성화 페이지](https://messages.google.com/disable-chat)

순서는 간단해요.

```
페이지 접속 > 국가 선택 > 전화번호 입력 > 인증번호 받기 > 인증번호 입력 > 확인
```

RCS를 해제한 다음에는 갤럭시 쓰는 친구한테 문자 한 번 보내달라고 해보세요. 새 아이폰으로 오면 해결된 거예요.

## 통화와 문자 둘 다 이상하다면 유심·개통 상태를 먼저 확인하세요

문자만 이전 폰으로 가는 게 아니라 새 아이폰에서 전화까지 안 된다면, 이건 RCS 문제가 아니에요. 유심 인식이나 개통 상태가 제대로 안 된 케이스예요.

유심을 옮겨 꽂는 것만으로는 안 되는 경우가 있거든요. 통신사에 단말기 변경 등록까지 돼야 정상적으로 개통이 완료될 수 있어요.

이럴 땐 아래 순서로 확인해보세요.

```
1. 새 아이폰에 유심이 제대로 들어가 있는지 확인
2. 새 아이폰에서 통화와 데이터가 되는지 확인
3. 예전 갤럭시폰에 유심이 아직 남아있는 건 아닌지 확인
4. 새 아이폰 재부팅
5. 비행기 모드 10초 켰다가 끄기
6. 그래도 안 되면 통신사 고객센터에서 기기 변경 후 신호 갱신 상태 확인
```

자급제 아이폰이라면 통신사에서 단말기 등록 상태를 다시 확인해야 할 수도 있어요. 통화랑 문자 둘 다 이상하면 아이폰 설정보다 통신사 쪽 개통 상태를 먼저 확인하는 게 훨씬 빨라요.

## 갤럭시에서 아이폰으로 바꿨는데 문자 안옴 인증번호만 안 온다면 메시지 필터와 통신사 스팸 차단을 확인해보세요

지인 문자는 잘 오는데 은행이나 카드, SNS 인증번호만 쏙 빠진다면, 새 아이폰의 메시지 필터나 통신사 스팸 차단 쪽을 봐야 해요.

먼저 메시지 앱에서 **알 수 없는 발신자, 스팸, 정크** 탭에 인증 문자가 따로 들어가 있는지 확인해보세요. 설정에서는 아래 경로로 들어가면 돼요.

```
설정 > 앱 > 메시지 > 알 수 없는 발신자 필터링
```

iOS 버전에 따라 아래 경로로 보일 수도 있어요.

```
설정 > 메시지 > 알 수 없는 발신자 필터링
```

거기도 없다면 통신사 스팸 차단 설정을 확인해보세요. 일반 문자는 오는데 인증번호만 안 올 때는 통신사 단계에서 걸러지는 경우가 있거든요.

## 갤럭시에서 아이폰으로 바꿨는데 문자 안옴 마치며

언니 경우는 딱 RCS 문제였어요.

갤럭시 친구들 문자가 새 아이폰이 아니라 예전 갤럭시폰으로 계속 오는 거예요. 예전 갤럭시에 RCS가 켜진 채로 남아있었던 거였고, 상대방 갤럭시가 언니 번호를 아직 갤럭시 사용자로 인식하고 있었던 거죠.

예전 갤럭시에서 RCS를 꺼주고 나서야 새 아이폰으로 문자가 제대로 오기 시작했어요. 같은 갤럭시에서 아이폰으로 바꿨는데 문자 안옴 상황이시라면 위에 정리한 순서대로 한번 확인해보세요.
