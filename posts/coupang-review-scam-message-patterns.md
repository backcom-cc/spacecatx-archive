## 진짜 쿠팡체험단은 어떻게 연락이 올까요?

쿠팡체험단 안내는 보통 아래 경로로 연락이 옵니다.

* 쿠팡 앱 알림
* 마이쿠팡 안의 체험단 관련 메뉴
* 쿠팡 공식 고객센터 번호(1577-7011)로 온 문자
* 쿠팡에 가입한 이메일

진짜 쿠팡체험단은 기본적으로 **쿠팡 앱 안에서 확인**할 수 있어요. 문자나 이메일로 안내가 오더라도, 결국 쿠팡 앱에서 확인되는지가 중요합니다.

문자 속 링크를 바로 누르지 말고, 직접 쿠팡 앱을 열어보세요. 앱에서 확인되면 진짜일 가능성이 높고, 카카오톡이나 외부 링크로 넘어가라고 하면 사기일 가능성이 높습니다.

## “쿠팡체험단”이라는 말, 사실 다 쿠팡 공식은 아닐 수 있어요

먼저 이것부터 구분해야 합니다. 문자나 광고에서 “쿠팡체험단”이라고 적혀 있어도, 전부 쿠팡이 직접 운영하는 공식 체험단은 아닐 수 있어요.

사람들이 말하는 쿠팡체험단에는 보통 세 가지가 섞여 있습니다.

| 구분 | 뜻 | 확인 기준 |
| --- | --- | --- |
| **1. 진짜 쿠팡체험단** | 쿠팡이 직접 운영하는 공식 체험단 | 쿠팡 앱 안에서 확인됩니다 |
| **2. 협찬/판매자 체험단** | 판매자나 마케팅 업체가 제품 리뷰를 부탁하는 경우 | 블로그, 인스타, 이메일 등 외부 채널로 제안이 올 수 있습니다 |
| **3. 사칭 사기** | 위 두 가지처럼 꾸며서 돈이나 개인정보를 노리는 문자 | 개인번호, 카톡, 텔레그램, 선결제, 환급 안내가 붙습니다 |

### 업체에서 요청하는 협찬 제품도 “쿠팡체험단”처럼 부르는 경우가 있어요

여기서 제일 헷갈리는 게 2번이죠. 쿠팡 공식 체험단 말고도, 판매자나 마케팅 업체가 제품 리뷰를 부탁하는 경우는 실제로 있을 수 있어요.

블로그나 인스타를 운영하는 사람에게 “제품 보내드릴 테니 후기 작성해주실 수 있나요?” 하고 제안하는 식이요. 이때 업체 쪽에서 “쿠팡 리뷰 체험단”, “쿠팡 상품평 체험단” 같은 말을 쓰기도 하는데, 이건 쿠팡이 직접 운영하는 공식 체험단이 아닙니다.

이런 협찬 제안은 보통 블로그, 인스타, 이메일처럼 내가 공개한 채널로 먼저 연락이 와요. 판매자가 리뷰어를 찾아서 제안하는 방식이기 때문에, 구매는 제안을 받고 나서 이뤄지는 경우가 많습니다.

그런데 개인 휴대폰 번호로 갑자기 연락이 왔다면, 내 번호를 어떻게 알았는지부터 생각해봐야 해요.

**쿠팡은 판매자에게 구매자의 실제 번호를 제공하지 않아요. 배송 과정에서 필요한 경우 안심번호만 제공하는데, 이마저도 배송 완료 후 48시간이 지나면 자동으로 폐기됩니다.**

그래서 “쿠팡에서 구매하셨기 때문에 연락드립니다”는 말 자체가 말이 안 됩니다.

> **내 번호를 어떻게 알고 연락한 거지?**  
> 꼭 의심해보세요.

## 쿠팡체험단 문자, 이것부터 확인하세요

쿠팡체험단 문자가 진짜인지 헷갈리면, 문자 내용보다 먼저 아래 항목을 보세요. 하나라도 오른쪽에 해당하면 링크를 누르지 않는 게 안전합니다.

| 체크할 것 | 🟢 안전한 쪽 | 🔴 의심해야 하는 쪽 |
| --- | --- | --- |
| **발신번호** | 쿠팡 공식 고객센터 번호 1577-7011 | 개인 휴대폰 번호, 모르는 번호 |
| **확인 위치** | 쿠팡 앱 알림, 마이쿠팡에서 확인됨 | 문자 링크에서만 확인됨 |
| **이동 경로** | 쿠팡 앱 안에서 진행 | 카카오톡, 오픈채팅, 텔레그램, 외부 사이트로 이동 |
| **결제 방식** | 업체 정보와 페이백 조건이 명확함 | 개인번호·카톡에서 먼저 주문 후 환급을 안내함 |
| **개인정보 요구** | 쿠팡 계정 안에서 처리 | 계좌번호, 카드정보, 신분증 요구 |
| **압박 문구** | 일반 안내 | 오늘 마감, 선착순, 즉시 참여, 한정 수량 |

특히 **개인번호 + 카카오톡 유도 + 선결제 후 환급**이 같이 나오면 거의 사칭 문자로 봐도 됩니다. 진짜인지 궁금하면 문자 속 링크 말고, 쿠팡 앱을 직접 열어서 확인하세요.
