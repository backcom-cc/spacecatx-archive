# 유튜브 “재생 ID – 오디오 렌더러 오류” 뜨면서 소리 안 날 때 해결법

## 유튜브는 재생되는데 소리만 안 나올 때

PC로 유튜브를 보다가 갑자기 이런 문구가 뜬 적 있으신가요?

> 재생 ID: XXXXXXXX  
> 오디오 렌더러 오류입니다. 컴퓨터를 다시 시작해 주세요.

영상은 나오는데 소리만 안 납니다. 이게 은근히 사람을 당황하게 만들어요. 스피커가 고장 난 건가 싶고, 이어폰 문제인가 싶고, 괜히 유튜브까지 새로고침하게 됩니다.

그런데 **유튜브 오디오 렌더러 오류**는 대부분 유튜브 영상 자체 문제라기보다, 컴퓨터가 “소리를 어디로 내보낼지” 헷갈린 상태에서 생기는 경우가 많습니다.

특히 블루투스 이어폰, 외장 모니터, HDMI, AirPods, USB 오디오 장치를 번갈아 쓰는 환경이라면 더 자주 생길 수 있습니다.

---

## 왜 유튜브 오디오 렌더러 오류가 뜰까요?

이 오류는 쉽게 말하면 오디오 출력 경로가 꼬였을 때 나타나는 증상에 가깝습니다. 영상은 정상적으로 불러왔는데, 소리를 내보낼 장치를 운영체제가 제대로 잡지 못하는 상황입니다.

주로 아래 상황에서 많이 발생합니다.

* 블루투스 이어폰을 연결했다가 끊은 직후
* 외장 모니터나 HDMI를 연결한 뒤 출력 장치가 바뀐 경우
* 절전 모드에서 복귀한 직후
* 윈도우 업데이트 이후 오디오 장치가 꼬인 경우
* 맥에서 AirPods 자동 전환이 이상하게 잡힌 경우
* USB 오디오 장치나 허브를 연결했다가 뺀 경우

그러니까 처음부터 스피커 고장으로 볼 필요는 없습니다. 먼저 출력 장치부터 다시 잡아보는 게 좋습니다.

---

## 유튜브 오디오 렌더러 오류 뜰 때 먼저 공통으로 해볼 것

윈도우든 맥이든, 제일 먼저 할 일은 간단합니다. 현재 소리가 어느 장치로 나가고 있는지 확인하는 것입니다.

1. 유튜브 탭을 새로고침합니다.
2. 소리 출력 장치를 다른 장치로 바꿨다가 다시 원래 장치로 돌립니다.
3. 블루투스 이어폰을 사용 중이라면 잠깐 연결을 끊습니다.
4. 외장 모니터나 HDMI를 쓰고 있다면 출력 장치가 모니터로 바뀌었는지 확인합니다.
5. 그래도 안 되면 브라우저를 완전히 종료했다가 다시 실행합니다.

생각보다 이 단계에서 해결되는 경우가 많습니다. 컴퓨터가 “아, 소리 여기로 내보내면 되는구나” 하고 다시 잡는 느낌이에요.

---

## 윈도우에서 해결하는 방법

### 1. 출력 장치를 다시 선택하기

윈도우에서는 작업 표시줄의 스피커 아이콘을 눌러 현재 출력 장치를 확인하세요.

* 스피커
* 헤드셋
* 블루투스 이어폰
* 외장 모니터 스피커
* USB 오디오 장치

다른 장치로 한 번 바꿨다가 다시 원래 쓰던 장치로 돌려보세요. 외장 모니터를 연결한 뒤라면 소리가 모니터 쪽으로 빠져 있을 수도 있습니다.

### 2. 블루투스 장치를 완전히 끊고 테스트하기

블루투스 이어폰이나 헤드셋을 사용 중이었다면, 블루투스를 잠깐 꺼보세요. 이어폰은 끊긴 것처럼 보이는데 윈도우가 여전히 해당 장치를 출력 장치로 잡고 있는 경우가 있습니다.

```
설정 > Bluetooth 및 장치 > Bluetooth 끄기
또는 작업 표시줄 빠른 설정 > Bluetooth 끄기
```

그 상태에서 유튜브를 다시 재생해보고, 노트북 스피커나 기본 스피커에서 소리가 나는지 확인합니다.

### 3. 오디오 드라이버 다시 잡기

윈도우 업데이트 이후부터 계속 오류가 난다면 오디오 드라이버가 꼬였을 수 있습니다. 이때는 장치 관리자에서 오디오 장치를 제거한 뒤 재부팅해볼 수 있습니다.

```
장치 관리자 > 사운드, 비디오 및 게임 컨트롤러 > 오디오 장치 우클릭 > 장치 제거 > 재부팅
```

재부팅하면 윈도우가 기본 오디오 드라이버를 다시 잡는 경우가 많습니다. 다만 회사 PC나 특수 오디오 장비를 쓰는 경우에는 드라이버 제거 전에 신중하게 확인하세요.

### 4. 그래도 안 되면 재부팅하기

오디오 렌더러 오류 메시지 자체가 “컴퓨터를 다시 시작해 주세요”라고 안내하는 경우도 있습니다. 실제로 윈도우에서는 재부팅 한 번으로 오디오 장치 충돌이 풀리는 경우가 많습니다.

급한 작업 중이라면 출력 장치 전환부터 해보고, 계속 반복된다면 재부팅으로 정리하는 쪽이 빠릅니다.

## 맥에서 해결하는 방법

맥은 윈도우처럼 오디오 드라이버를 따로 지우고 다시 설치하는 방식보다는, 출력 장치를 다시 잡거나 블루투스 연결을 정리하는 방식이 더 중요합니다.

### 1. 사운드 출력 장치 다시 선택하기

먼저 시스템 설정에서 사운드 출력 장치를 확인하세요.

```
시스템 설정 > 사운드 > 출력
```

여기에서 현재 선택된 장치를 확인합니다.

* MacBook 스피커
* AirPods
* 외장 모니터
* USB 오디오 장치
* 블루투스 스피커

다른 장치로 바꿨다가 다시 원래 장치로 돌려보세요. 맥은 AirPods나 외장 모니터 사이에서 출력 장치가 꼬이는 일이 꽤 있습니다.

### 2. 블루투스를 껐다 켜기

AirPods를 쓰다가 케이스에 넣었는데도 맥이 여전히 연결된 것처럼 잡고 있는 경우가 있습니다. 이럴 때는 블루투스를 완전히 껐다가 다시 켜보세요.

```
시스템 설정 > Bluetooth > Bluetooth 끄기 > 몇 초 대기 > 다시 켜기
```

그다음 유튜브를 새로고침하고 소리가 정상으로 돌아오는지 확인합니다.

### 3. Audio MIDI 설정 확인하기

맥에서 유튜브만 소리가 이상하거나 특정 앱에서만 소리가 안 난다면 Audio MIDI 설정도 확인해볼 수 있습니다.

```
응용 프로그램 > 유틸리티 > Audio MIDI 설정
```

사용 중인 출력 장치를 선택한 뒤 샘플레이트가 너무 특이하게 설정되어 있지 않은지 확인하세요. 일반적으로는 44.1kHz 또는 48kHz로 맞춰두면 무난합니다.

전문 오디오 장비를 쓰지 않는 일반 사용자는 이 설정을 자주 만질 필요는 없지만, 외장 오디오 장치를 쓰다가 소리가 꼬였을 때는 확인해볼 만합니다.

### 4. 맥도 결국 재시동이 빠를 때가 많습니다

맥은 구조상 오디오 드라이버를 직접 재설치하는 방식이 아니라, 출력 장치 재선택이나 재시동으로 문제를 정리하는 경우가 많습니다.

괜히 이것저것 복잡하게 만지기보다, 출력 장치와 블루투스를 확인한 뒤에도 계속 안 되면 재시동이 가장 빠른 해결책일 수 있습니다.

## 유튜브 오디오 렌더러 오류 이런 경우는 유튜브 문제가 아닐 수 있습니다

오디오 렌더러 오류가 뜨면 유튜브가 문제인 것처럼 보이지만, 실제로는 컴퓨터의 오디오 장치 충돌인 경우가 많습니다.

* 다른 브라우저에서도 소리가 안 남
* 유튜브뿐 아니라 다른 사이트 영상도 소리가 안 남
* 음악 앱이나 게임에서도 소리가 안 남
* 출력 장치가 갑자기 모니터나 블루투스 장치로 바뀜

이런 경우에는 유튜브 설정만 만질 것이 아니라, 운영체제의 소리 출력 장치부터 확인해야 합니다.

반대로 유튜브에서만 문제가 생긴다면 브라우저 캐시, 확장 프로그램, 하드웨어 가속 문제도 함께 볼 수 있습니다.

YouTube 고객센터의 [동영상 오류 문제 해결 안내](https://support.google.com/youtube/answer/3037019)에서도 브라우저 업데이트, 캐시와 쿠키 삭제, 기기 재시작 등을 기본 해결 방법으로 안내합니다. 유튜브 오디오 렌더러 오류가 반복된다면 먼저 출력 장치와 기기 재시동을 확인해보세요.

## 정리

유튜브에서 “재생 ID – 오디오 렌더러 오류”가 뜨고 영상은 나오는데 소리만 안 난다면, 대부분은 오디오 출력 경로 충돌을 먼저 의심해보는 것이 좋습니다.

* 윈도우는 출력 장치 재선택, 블루투스 해제, 오디오 드라이버 재설치, 재부팅 순서로 확인
* 맥은 사운드 출력 장치 재선택, 블루투스 재연결, Audio MIDI 설정, 재시동 순서로 확인
* 블루투스 이어폰이나 외장 모니터를 자주 바꾸는 환경에서 더 자주 발생할 수 있음
* 영상은 나오는데 소리만 안 난다면 유튜브보다 운영체제 오디오 경로를 먼저 확인

**처음 보면 스피커가 고장 난 것 같지만, 대부분은 소리 출력 장치가 꼬인 문제입니다. 출력 장치를 다시 선택하고, 블루투스를 껐다 켜고, 그래도 안 되면 재시동부터 해보세요.**
