##### 타오바오 로그인 안됨 해결법

이놈의 타오바오는 가입도 힘든데 로그인 오류까지 나면 한자의 향연에 진짜 미쳐버릴 것 같죠. 저도 여기저기 알아보다가 겨우 해결해서, 비슷하게 막힌 분들 덜 헤매시라고 핵심만 정리해봤습니다.

> **한 줄 결론:** 한자 폭격에 당황하지 마세요. 타오바오 로그인 오류는 대부분 인증문자, 비정상 접근, 계정 제한 세 가지 안에서 해결 방향이 갈립니다. **일단 로그인은 앱으로, 이후 QR코드 인증으로 탐색은 PC로!**

## 먼저 내 상황부터 빠르게 확인해보세요

| 화면에 보이는 상황 | 먼저 의심할 것 | 가장 먼저 해볼 것 |
| --- | --- | --- |
| 인증문자가 안 옴 | 국제문자 차단, 재요청 과다 | 시간 두고 재시도, 앱 경로 확인 |
| 비정상 접근으로 보임 | VPN, 반복 로그인, 낯선 IP | VPN 끄기, LTE/5G로 전환 |

| 계정 제한/동결 | 로그인 정보 변경 반복, 접속 환경 변화 | 알리페이 인증 여부 확인 |
| 슬라이드 인증 반복 | 브라우저 쿠키, PC 환경 꼬임 | 시크릿 모드, 앱 QR 로그인 |

## 타오바오 로그인 안됨 상황별 해결방법

### 1. 인증문자가 안 옵니다 (验证码, 发送失败, 短信验证)

국제문자 차단, 짧은 시간 안의 반복 요청, 서버 지연 때문에 가장 자주 막히는 유형입니다.

* 통신사 앱이나 고객센터에서 **국제문자 수신 차단** 여부를 먼저 확인해보세요.
* 인증번호 재전송 버튼을 연속으로 누르지 마세요.
* 1분 뒤 재시도보다 **30분~1시간 뒤 다시 시도**하는 쪽이 낫습니다.
* 문자 대신 **알리페이 앱을 켜둔 상태에서 로그인하면, 알리페이 앱으로 ‘승인하시겠습니까?’ 같은 푸시가 오고 이걸로 인증을 대체할 수도 있습니다.**
* PC만 붙잡지 말고 모바일 앱 로그인도 같이 시도해보세요.

### 2. 비정상 접근으로 보입니다 (异常, 非正常活动, 异常流量)

짧은 시간 안에 로그인 반복, VPN 사용, 새 기기 접속 때문에 보안 차단이 걸린 상태입니다.

* VPN을 바로 끄세요.
* 와이파이 대신 **LTE/5G**로 접속해보세요.
* PC 웹보다 **모바일 앱 로그인**을 먼저 시도해보세요.
* 앱에서 로그인한 뒤 PC는 **QR 스캔**으로 들어가보세요.
* **여러 번 연속 로그인하지 말고, 여러 창도 동시에 띄우지 마세요.**

### 3. 로그인 환경이 위험하다고 나옵니다 (登录环境存在风险, 安全风险, 验证身份)

새 기기, 새 위치, 다른 네트워크에서 접속하면 타오바오가 본인 확인을 다시 요구할 수 있습니다.

* 기존에 로그인된 다른 기기가 있다면 그 앱의 **보안센터(安全中心)**에서 새 기기 접근 허용 또는 로그인 승인 항목을 먼저 확인해보세요.
* 모바일 앱에서 승인 또는 인증 절차를 먼저 진행해보세요.
* **알리페이 앱에서 여권 인증(L3)이 완료되어 있으면 통과가 가능하기도 합니다.**
* 새 폰으로 바꾼 직후라면 PC보다 앱에서 먼저 로그인해보세요.
* 로그인에 쓰는 기기, 네트워크, 방식을 자주 바꾸지 않는 편이 좋습니다.

### 4. 계정이 제한되었거나 동결된 상태입니다 (严重违规, 限制登录, 账号异常)

로그인 수단, 기기, 접속 환경이 자주 바뀌면 타오바오가 계정을 위험하게 보고 제한을 걸 수 있습니다. 이 단계에서는 재로그인 반복으로는 잘 풀리지 않습니다.

* 로그인 방식, 기기, 네트워크를 한동안 **고정해서 사용**해보세요.
* 알리페이 쪽 **본인 인증 상태**를 먼저 확인해보세요.
* 필요하면 알리페이 고객센터나 계정 인증 절차를 통해 **여권 인증**을 진행해보세요.
* `严重违规`처럼 강한 문구가 보이면 일반 재시도보다 **계정 복구 절차**를 먼저 확인하는 편이 낫습니다.
* 계속 막히면 고객센터 문의도 같이 고려해보세요. 심각한 위반(严重违规)이 있다면 전화가 빠를 수도 있어요. (홍콩 센터는 영어 상담이 가능합니다!)

### 5. 슬라이드 인증이 계속 반복됩니다 (滑块验证, 请按住滑块)

브라우저 쿠키 충돌, 확장 프로그램 간섭, 접속 환경 문제로 자주 발생합니다.

* 크롬이나 엣지 **시크릿 모드**에서 다시 시도해보세요.
* 다른 브라우저로 바꿔보세요.
* 브라우저 쿠키와 캐시를 삭제해보세요.
* PC에서 계속 반복되면 **모바일 앱 로그인**으로 우회해보세요.
* 앱에서 로그인한 뒤 **QR 로그인**으로 PC에 들어가는 방법도 추천합니다.

### 6. 비밀번호가 틀리거나 계정이 없다고 나옵니다 (密码错误, 账号不存在)

아이디 형식 오류, 번호 입력 방식 차이, 가입 수단 착각 때문에 많이 생깁니다.

* 휴대폰 번호 입력 형식이 조금만 달라도 `계정 없음`이나 `비밀번호 오류`가 뜰 수 있습니다.
* 한국 번호는 국가코드 **82**를 선택한 뒤, 휴대폰 번호 앞자리 **0을 제외하고 입력하는 방식**부터 먼저 확인해보세요.  
  (`예: 010-1234-4567 → 1012344567`)
* 번호 로그인이 안 되면 가입 당시 사용한 **이메일 전체 주소**로도 다시 시도해보세요.
* 애플, 구글 등 소셜 로그인으로 가입했다면 **가입했던 방식 그대로** 다시 로그인해보세요. **그리고 자꾸 로그인 수단을 변경하지 마시고 하나로만 사용하세요** (제가 이것때문에 많이 막혔습니다)
* 그래도 안 되면 비밀번호 재설정을 진행해보세요.
