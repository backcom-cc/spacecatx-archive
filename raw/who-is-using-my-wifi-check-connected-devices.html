우리집 와이파이 누가 쓰는지 확인할 수 있을까?
집에서 인터넷이 갑자기 느려질 때 이런 생각 한 번쯤 해보셨죠?

“혹시… 우리 집 와이파이 누가 몰래 쓰는 거 아니야?”
“와이파이 도둑놈이 있나봐..”

특히 아파트나 원룸, 오피스텔에서는 같은 건물 사람들이 많다 보니 괜히 더 의심이 됩니다.

결론부터 말하면 집 와이파이를 누가 쓰고 있는지는 어느 정도 확인할 수 있습니다.
다만 많은 사람들이 생각하는 것처럼 인터넷 사용 기록까지 다 보이는 것은 아닙니다.

한줄 결론
공유기 관리 페이지나 Fing 앱으로 현재 연결된 기기는 확인 가능합니다. 인터넷 기록까지 보이는 건 아니지만, 모르는 기기가 있는지는 바로 알 수 있어요.

1. 공유기 관리 페이지 접속 (가장 확실한 방법)

공유기의 설정 페이지에 직접 들어가서 현재 연결된 기기 목록을 확인하는 방법입니다.

브라우저 주소창에 아래 주소 중 하나를 입력합니다.

192.168.0.1
192.168.1.1
192.168.219.1 (KT 공유기)

공유기 브랜드마다 주소는 조금씩 다르지만 대부분 위 주소 중 하나입니다.

로그인 후 아래 메뉴에서 확인할 수 있습니다.

[관리도구]
→ [고급 설정]
→ [네트워크 관리]
→ [내부 네트워크 설정]
또는
[무선 접속 정보]

여기서 현재 와이파이에 연결된 기기 목록을 확인할 수 있습니다.

팁
내 폰, 노트북, 태블릿, TV 같은 익숙한 기기 말고 모르는 기기 이름이 보이면 와이파이 비밀번호가 유출됐을 가능성도 있습니다.

2. 스마트폰 앱으로 확인하는 방법 (가장 간편)

공유기 관리자 페이지 접속이 어렵다면 스마트폰 앱으로도 확인할 수 있습니다.

가장 많이 사용하는 앱이 바로 Fing입니다.

앱 이름 : Fing
지원 : iOS / 안드로이드

사용 방법은 간단합니다.

  • 앱 설치
  • 장치 스캔 실행
  • 현재 와이파이에 연결된 기기 목록 확인

기기 브랜드도 표시됩니다.

Samsung
Apple
LG
Windows PC

그래서 공유기보다 어떤 기기인지 더 쉽게 파악되는 경우도 많습니다.

3. PC 프로그램으로 확인하는 방법

PC에서 확인하고 싶다면 이런 프로그램도 있습니다.

Wireless Network Watcher

특징은 다음과 같습니다.

  • 설치 없이 실행 가능
  • 현재 네트워크 연결 기기 자동 스캔
  • IP / MAC 주소 표시
  • 실시간 갱신

특히 집에서 PC를 많이 쓰는 분들은 이 방법이 편합니다.

4. 부모님이 공유기 보면 인터넷 기록까지 볼 수 있을까?

많이 궁금해하는 질문입니다.

“부모님이 공유기 보면 내가 어떤 사이트 봤는지 알 수 있나요?”

결론은 일반 가정용 공유기로는 대부분 불가능합니다.

공유기에서 확인 가능한 정보는 보통 이 정도입니다.

현재 연결된 기기
IP 주소
MAC 주소
기기 이름

하지만 아래 같은 정보는 대부분 확인할 수 없습니다.

어떤 사이트 접속
검색 기록
사용한 앱

모르는 기기가 보였다면 대처 방법

정체 모를 기기가 보이면 그냥 두지 말고 바로 조치하는 것이 좋습니다.

  • 와이파이 비밀번호 변경
  • 공유기 관리자 비밀번호 변경
  • 보안 방식 WPA2 또는 WPA3 사용
  • MAC 주소 인증 설정

특히 MAC 주소 인증을 사용하면 등록된 기기만 접속할 수 있어서 보안이 훨씬 강해집니다.

우리집 와이파이 누가 쓰는지 확인 할 수 있나? 정리

우리 집 와이파이를 누가 쓰고 있는지 확인하려면 다음 방법을 사용하면 됩니다.

1. 공유기 관리자 페이지 접속
2. 스마트폰 네트워크 스캔 앱 사용
3. PC 네트워크 감지 프로그램 사용

다만 일반 가정용 공유기는 누가 연결되어 있는지 정도만 확인 가능하고,
인터넷 사용 기록까지 자세히 보는 기능은 대부분 없습니다.

그래도 한 번쯤 확인해보면 의외로 집 안 기기들이 잔뜩 붙어 있는 경우가 많습니다. 생각보다 TV, 로봇청소기, 태블릿까지 다 와이파이에 붙어 있거든요.
