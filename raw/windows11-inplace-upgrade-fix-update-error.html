윈도우 업데이트 자체가 꼬인 상태입니다.

장기 대수술이 필요한 단계입니다.

최후의 해결 방법: 인플레이스 업그레이드

이름은 어렵지만 개념은 간단합니다.

윈도우를 새로 설치하면서도,
개인 파일과 프로그램은 그대로 유지하는 방식

즉, 포맷 없이 윈도우만 덮어씌운다고 보면 됩니다.

이 방법을 써야 하는 상황

  • 윈도우 업데이트 오류가 반복될 때
  • 0x800 계열 오류가 계속 발생할 때
  • sfc / dism 실행 후에도 해결되지 않을 때
  • 시스템 전반이 불안정한 느낌이 들 때

위 조건에 해당하면 인플레이스 업그레이드를 고려할 단계입니다.

준비물

  • 인터넷 연결
  • 윈도우 11 설치 도구

→👉 Windows 11 설치 도구 공식 다운로드 페이지 바로가기

Microsoft 공식 페이지에서 다운로드 후 실행

진행 방법

  1. 설치 도구 실행
  2. “이 PC 업그레이드” 선택
  3. “개인 파일 및 앱 유지” 옵션 선택
  4. 설치 진행

설치 중에는 여러 번 재부팅이 진행되며, 보통 30분에서 1시간 정도 소요됩니다.

주의사항

  • 설치 중 전원을 끄면 시스템 손상 가능
  • 중요 데이터는 사전 백업 권장
  • 노트북은 반드시 전원 연결 상태 유지

정리

여기까지 왔다는 것은 이미 기본적인 해결 방법은 모두 시도한 상태입니다.

인플레이스 업그레이드는 대부분의 업데이트 오류를 해결할 수 있는 방법입니다.

이 방법으로도 해결되지 않는다면, 포맷까지 고려해야 하는 단계입니다.

인플레이스 설치 중 혹시 모를 상황에 대비해 중요 데이터는 반드시 외장 SSD나 클라우드에 백업해두세요. 요즘은 속도 빠른 NVMe 외장 하드가 필수가 됐네요.
