Apple ID에 신뢰하는 전화번호가 추가되었습니다? 해킹인지 확인하는 방법

야밤에 갑자기 이런 알림이 왔습니다.

“Apple ID에 신뢰하는 전화번호가 추가되었습니다.”

순간 멈췄습니다.
최근에 아무것도 추가한 기억이 없었거든요.

새 기기에 로그인한 것도 아니고, 보안 설정을 건드린 적도 없었습니다.
그런데 갑자기 이런 알림이 오니까 솔직히 좀 무서웠습니다.

“설마 해킹인가?”

결론부터 말하면 이 알림이 무조건 해킹이라는 뜻은 아닙니다.
하지만 내가 추가한 기억이 없다면 그냥 넘기면 안 됩니다.

신뢰하는 전화번호 목록을 열어서 모르는 번호가 있는지 먼저 확인하세요.

내가 추가한 기억이 없다면 문자·메일 링크는 누르지 말고, 설정 앱에서 신뢰하는 전화번호 목록부터 확인하세요.

애플 계정 신뢰하는 전화번호 알림은 무슨 뜻일까?

Apple ID에는 신뢰하는 전화번호라는 보안 정보가 있습니다.

이 번호는 Apple ID에 로그인하거나, 비밀번호를 바꾸거나, 계정을 복구할 때 인증 코드를 받을 수 있는 번호입니다.

그래서 신뢰하는 전화번호가 추가되면 Apple이 알림을 보냅니다.
계정 보안에 중요한 변경이 있었기 때문입니다.

내가 직접 번호를 추가했다면 정상 알림이지요.
하지만 추가한 기억이 없는데 알림이 왔다면 바로 확인해야 합니다.

먼저 문자나 메일 링크를 누르지 마세요

이 알림이 문자나 메일로 왔다면 가장 먼저 조심해야 할 게 있습니다.

메시지 안의 링크를 누르지 마세요.

진짜 Apple 보안 알림처럼 보여도, 링크가 피싱 사이트로 연결될 수 있습니다.

확인은 링크로 하지 말고 직접 들어가세요.

아이폰에서는 설정 앱에서 확인합니다.

설정 > Apple ID 이름 > 로그인 및 보안 > 이중 인증 > 신뢰하는 전화번호

웹에서는 Apple 공식 계정 페이지로 직접 접속합니다.

account.apple.com > 로그인 및 보안 > 계정 보안 > 신뢰하는 전화번호

주소를 검색해서 들어가거나 직접 입력하세요.
문자나 메일에 들어 있는 버튼은 누르지 않는 게 안전합니다.

아래 기준으로 먼저 구분하세요.

가장 중요한 기준은 신뢰하는 전화번호 목록에 모르는 번호가 있는지입니다.

모르는 번호가 없다면 일단 큰 위험은 줄어듭니다.
모르는 번호가 있다면 바로 삭제하고 Apple ID 비밀번호를 변경하세요.

신뢰하는 전화번호 확인 방법

아이폰에서는 설정 앱에서 확인할 수 있습니다.

설정 > Apple ID 이름 > 로그인 및 보안 > 이중 인증 > 신뢰하는 전화번호

여기에서 현재 Apple ID에 등록된 신뢰하는 전화번호를 확인하세요.

내 번호만 있다면 일단 괜찮습니다.
하지만 모르는 번호가 있다면 바로 삭제해야 합니다.

모르는 번호가 있으면 바로 삭제하세요

신뢰하는 전화번호 목록에 모르는 번호가 있다면 그냥 두면 안 됩니다.

그 번호로 Apple ID 인증 코드가 갈 수 있습니다.
계정 복구나 로그인 인증에도 영향을 줄 수 있어요.

이 경우에는 바로 아래 순서대로 처리하세요.

  1. 모르는 신뢰하는 전화번호 삭제
  2. Apple ID 비밀번호 변경
  3. 로그인된 기기 목록 확인
  4. 모르는 기기 제거
  5. 계정 정보가 바뀐 부분이 있는지 확인

번호만 지우고 끝내면 부족합니다.
비밀번호와 로그인된 기기까지 같이 확인해야 합니다.

Apple ID 비밀번호를 변경하세요

모르는 번호가 등록되어 있었다면 비밀번호를 바로 바꾸세요.

예전에 쓰던 비밀번호와 비슷한 조합은 쓰지 마세요.
다른 사이트에서 쓰던 비밀번호도 쓰면 안 됩니다.

새 비밀번호는 Apple ID 전용으로 만들어야 합니다.

비밀번호 변경은 아이폰 설정에서 할 수 있습니다.

설정 > Apple ID 이름 > 로그인 및 보안 > 암호 변경

비밀번호를 바꾼 뒤에는 다시 신뢰하는 전화번호 목록을 확인하세요.
모르는 번호가 삭제된 상태인지 확인해야 합니다.

로그인된 기기 목록도 확인하세요

신뢰하는 번호만 확인하면 끝이 아닙니다.

Apple ID에 모르는 기기가 로그인되어 있는지도 봐야 합니다.

아이폰 설정 앱 맨 위에서 Apple ID 이름을 누르고, 화면 아래쪽으로 내려가면 로그인된 기기 목록이 나옵니다.

여기에서 모르는 iPhone, iPad, Mac이 보이면 계정에서 제거하세요.

설정 > Apple ID 이름 > 기기 목록 > 모르는 기기 선택 > 계정에서 제거

내가 쓰지 않는 기기가 연결되어 있다면 계정 접근이 있었을 수 있습니다.
번호 삭제와 비밀번호 변경만 하고 끝내지 말고, 기기 목록까지 확인하세요.

아이폰이 없어도 웹에서 확인할 수 있습니다

아이폰이 없거나 당장 설정 앱을 열 수 없다면 웹에서 확인하세요.

브라우저에서 Apple 공식 계정 페이지로 직접 접속하면 됩니다.

account.apple.com > 로그인 및 보안 > 계정 보안

여기에서 신뢰하는 전화번호, 계정 보안 정보, 로그인 관련 정보를 확인할 수 있습니다.

다시 말하지만 문자나 메일 링크를 누르지 마세요.
브라우저 주소창에 직접 입력해서 들어가는 게 안전합니다.

  • 새 아이폰, 아이패드, 맥에 로그인한 경우
  • Apple ID 보안 설정을 직접 변경한 경우
  • 전화번호를 새로 추가한 경우
