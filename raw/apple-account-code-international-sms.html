국제발신 Apple 계정 코드
이 문자가 오는 상황은 하나입니다

Apple 인증 코드는 아무한테나 날아오는 문자가 아닙니다.

아이디와 비밀번호, 이 두 가지를 입력한 다음 단계에서 요구되는 코드입니다.

그러니까 이 문자가 왔다는 건, 누군가 이미 내 애플계정으로 로그인 절차를 진행했고 2단계 인증 단계까지 들어왔을 가능성이 높다는 뜻입니다.

갤럭시 쓰는데 왜 이 문자가 오냐고요?

Apple 계정은 아이폰 안에만 있는 계정이 아닙니다. 웹 브라우저에서도 로그인할 수 있고, 예전에 계정을 만들었거나 한 번이라도 쓴 적 있으면 그 계정은 살아있는 상태일 수 있습니다.

지금 갤럭시를 쓰고 있어도, 내 Apple 계정이 있는 이상 이 문자는 올 수 있습니다.

아이폰에서 로그아웃했다고 문자가 멈추지도 않습니다.

내 기기에서 계정을 뺀 것과, 누군가 밖에서 Apple 계정 로그인을 시도하는 건 완전히 다른 일이기 때문입니다.

그럼 지금 해킹당한 건가요?

일단 한숨 놓으셔도 됩니다. 코드 단계까지 왔는데 코드를 못 받으니까, 거기서 막혀서 계속 시도하고 있는 상황일 가능성이 커요.

지금 이 순간은 2단계 인증이 버텨주고 있는 거예요. 근데 여기서 “다행이다” 하고 폰 내려놓으면 안 됩니다!
2단계 인증 코드가 왔다는 건, 비밀번호 입력 단계는 이미 넘어왔을 가능성이 있다는 뜻입니다. 계정이 완전히 털린 건 아니지만, 비밀번호는 노출됐을 가능성을 의심해야 합니다.

지금은 2단계 인증이 마지막 문 하나를 잡고 있는 상황인거에요.

지금 바로 해야 할 것들

1. 비밀번호부터 바꾸세요

순서 따질 것 없이 이게 제일 먼저입니다.

설정 > 내 이름 > 로그인 및 보안 > 암호 변경

지금 쓰는 비밀번호와 완전히 다른 비밀번호로 바꾸세요.

혹시 다른 사이트에서도 같은 비밀번호를 쓰고 있었다면, 그 사이트 비밀번호도 같이 바꾸는 게 좋습니다. 한 군데에서 유출된 비밀번호는 다른 계정 로그인 시도에도 사용될 수 있습니다.

2. 연결된 기기 목록을 확인하세요

비밀번호를 바꿨으면 바로 기기 목록을 확인하세요.

설정 > 내 이름 > 아래쪽 기기 목록

내가 쓰는 기기인지 하나씩 확인하세요.

팔았거나 잃어버린 기기, 이름이 낯선 기기가 보이면 바로 제거하세요. 모르는 기기가 연결돼 있다는 건 그냥 넘길 수 없는 상황입니다.

3. 신뢰하는 전화번호를 확인하세요

설정 > 내 이름 > 로그인 및 보안

2단계 인증 코드를 받을 수 있는 번호가 여기에 등록되어 있습니다.

내 번호가 맞는지, 모르는 번호가 끼어있지 않은지 확인하세요. 모르는 번호가 있으면 즉시 삭제해야 합니다.

애플 계정 신뢰하는 번호가 추가되었습니다? 해킹인지 확인 방법

4. 결제 내역도 훑어보세요

혹시 모를 금전적 피해를 막기 위해 앱스토어 결제나 구독 내역도 훑어보는 게 안전합니다. 비밀번호를 바꾼 김에 모르는 결제나 구독이 있는지 같이 확인하세요.

지금 코드 단계에서 막힌 거라면 실제 결제 피해는 없을 가능성이 크지만, 그래도 확인은 해두는 게 마음 편합니다.

5. 다른 기기에서 전부 로그아웃하세요

비밀번호를 변경할 때 “다른 기기에서 로그아웃” 옵션이 보이면 선택하세요.

코드 문자가 여러 번 왔거나, 모르는 기기가 보였다면 특히 해두는 게 좋습니다.

문자 차단보다 이게 훨씬 중요한 조치입니다.

6. 그래도 계속 온다면 Apple에 직접 연락하세요

비밀번호를 바꿨는데도 코드가 계속 날아온다면, 직접 문의하는 게 맞습니다.
