아이폰 Apple ID 기기 목록을 처음 제대로 들여다본 게 얼마 전이었어요.

요새 애플 보안 때문에 말이 많길래 들어갔다가 멈칫했습니다. 아이폰, 아이패드는 그렇다 치고 예전 회사 다닐 때 쓰던 아이맥, 진작에 팔아치운 아이패드, 거기다 Windows PC까지 달려있는 거예요.

엥, Windows? 애플 기기도 아닌데 형이 왜 여기서 나와? 뭐야?

모르는 기기가 있다는 게 해킹인지, 그냥 오래된 흔적인지 처음엔 구분이 안 됐어요. 그래서 정리해봤습니다.

Apple ID 기기 목록에 모르는 기기가 보인다고 바로 해킹은 아니지만, 전혀 기억나지 않는 기기라면 정체를 확인한 뒤 제거하고 비밀번호까지 점검하는 게 안전합니다.

아이폰 Apple ID 기기 목록이 뭔가요?

설정 > 맨 위 내 이름 > 아래쪽 기기 목록

여기 뜨는 게 “내가 산 애플 제품 목록”이 아니에요. 내 Apple ID로 로그인된 기기 목록입니다.

지금 쓰는 기기뿐 아니라 예전에 쓰던 기기, 다른 환경에서 로그인한 기록까지 잡혀 있을 수 있어요.

그래서 기기를 여러 개 쓴다면 아이폰, 아이패드, 맥북, 애플워치가 같이 뜨는 건 자연스러운 일입니다.

먼저 삭제하지 말고 정체부터 확인하세요

모르는 기기가 보인다고 바로 삭제부터 누르지는 마세요. 먼저 눌러서 기기 이름, 모델명, OS 버전을 확인해보세요.

설정 > 맨 위 내 이름 > 아래쪽 기기 목록 > 모르는 기기 선택

이름만 보면 남의 기기 같아도 실제로는 예전에 쓰던 내 기기일 수 있어요.

예전 아이폰, 가족 기기에 잠깐 로그인했던 기록, 중고로 팔기 전에 계정 정리가 덜 된 기기들이 여기에 남아있는 경우가 있습니다.

내 기기인 게 확인됐다면 굳이 삭제하지 않아도 됩니다. 대신 이름만 알아보기 쉽게 바꿔두면 다음에 덜 놀라요.

예를 들면 “iPad” 대신 “내 아이패드 미니”, “MacBook Pro” 대신 “회사 맥북” 이런 식으로요.

Windows PC가 왜 여기 있는 거죠?

이게 제일 의아하죠. 애플 기기도 아닌데 왜 뜨는 거야 싶은데, 이 목록이 “애플 제품 목록”이 아니라 “Apple ID 로그인 기기 목록”이라서 그렇습니다.

Windows에서도 Apple ID를 쓸 일이 생각보다 있습니다.

  • Windows용 iCloud를 설치해서 사진, 파일을 동기화한 경우
  • Apple Music Windows 앱에 로그인한 경우
  • 회사 PC나 공용 PC에서 iCloud.com에 로그인한 경우
  • Apple 계정 페이지에 접속했다가 로그아웃하지 않은 경우

잠깐 로그인하고 잊어버린 경우가 대부분입니다. 그래도 전혀 기억나지 않는 Windows PC라면 그냥 넘기지 말고 제거하고 비밀번호도 바꾸세요.

잘못 삭제하면 어떻게 되나요?

결론부터 말하면 아이폰이 초기화되거나 사진이 바로 날아가지는 않습니다.

다만 제거된 기기는 내 Apple ID 연결에서 빠지기 때문에 iCloud, 나의 찾기, 이중 인증 코드 표시 같은 기능을 더 이상 사용할 수 없게 됩니다.

내 기기였다면 다시 Apple ID로 로그인하면 되고, 모르는 기기였다면 다시 뜨지 않는 게 정상입니다.

삭제 자체를 너무 무서워할 필요는 없지만, 확인도 안 하고 무작정 누르는 건 추천하지 않습니다.

아이폰 Apple ID 기기 목록에 모르는 기기가 뜰 때,
이럴 땐 꼭 삭제해야 해요!

아래 상황이라면 오래 고민하지 말고 제거하는 게 안전합니다.

전혀 기억나지 않는 기기인데 최근 Apple ID 로그인 알림이나 인증 코드 문자까지 같이 왔다면 바로 제거하세요.

결제 내역이나 구독 내역에 모르는 항목이 있거나, 제거했는데 같은 기기가 다시 뜨거나, 일련번호가 내 기기와 맞지 않는다면 단순한 예전 기기 흔적이 아닐 수 있습니다.

이 경우는 기기를 제거하고 Apple ID 비밀번호도 같이 바꾸세요.

내 APPLE 계정에서 제거하는 방법

설정 > 맨 위 내 이름 > 기기 선택 > 계정에서 제거

제거하면 해당 기기는 내 Apple ID 연결에서 빠집니다.

내 기기였다면 다시 로그인하면 되고, 모르는 기기였다면 다시 뜨지 않는 게 정상입니다.

제거 후에도 찝찝하다면 비밀번호 변경과 결제 내역 확인까지 같이 해두세요.

Apple 공식 지원 문서에서도 Apple 계정에 로그인된 기기 목록을 확인하고, 필요 없는 기기를 제거할 수 있다고 안내하고 있습니다. 자세한 내용은 Apple 계정 기기 목록 확인 방법에서 확인할 수 있습니다.

기기 목록에 모르는 게 있다고 전부 해킹은 아닙니다.

예전 기기 흔적이거나 Windows용 iCloud 로그인 기록인 경우도 많아요.

하지만 그냥 둬야 할 이유도 없습니다. 정체를 확인하고, 내 기기면 이름을 정리하고, 모르는 기기면 제거하면 됩니다.

막상 들어가 보면 저처럼 팔아치운 기기, 회사 PC 흔적이 주렁주렁 달려있을 수 있으니까 가끔 한 번씩 정리해두는 걸 추천합니다.
