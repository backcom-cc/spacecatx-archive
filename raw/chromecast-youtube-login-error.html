크롬캐스트 유튜브 로그인 안 되나요? 다른 스마트TV 유튜브에도 되는 방식이에요.

얼마 전에 친구들이랑 펜션 놀러 갔는데요. 저녁 먹고 다 같이 거실에 모여서 TV로 유튜브나 틀어볼까 하고 리모컨을 잡았습니다.

요즘은 펜션 TV에도 대부분 유튜브 앱이 깔려 있잖아요. 그래서 그냥 제 계정으로 로그인하려고 했죠.

그런데 로그인을 했는데 로그인 화면이 계속 반복되거나 갑자기 튕겨버리는 겁니다. QR 로그인도 해보고, 휴대폰으로 연결도 해봤는데 로그인 > 다시 로그인 > 다시 로그인 무한 반복이었습니다.

그러다 TV 화면을 보니까 뭔가 수상한 흔적이 보이더라고요. 누군가 로그인해둔 계정, 알 수 없는 추천 목록, 여러 개의 로그인 기록. 여기 왔다 간 사람들이 계속 로그인했다가 그냥 나간 것 같았습니다.

이런 상황, 펜션이나 에어비앤비, 호텔 TV에서 꽤 자주 생깁니다. 여러 사람이 로그인했다가 로그아웃을 안 하고 나가면 로그인 기록이 뒤섞여서 인증이 꼬이거든요.

펜션·호텔 같이 단기간 머무르는 곳이라면 캐스팅이 제일 편합니다. 로그인 없이 내 폰 그대로 TV로 보낼 수 있고, 내 흔적도 안 남아요. 우리 집 TV라면 youtube.com/activate 로 한 번 제대로 연결해두는 게 낫습니다.

단기간 머무르는 곳 (펜션·호텔·에어비앤비)

방법 1. 캐스팅으로 보내기 (제일 간단)

TV 앱에 로그인할 필요 없이, 내 폰 유튜브 앱에서 영상 상단의 캐스팅 아이콘(정사각형에 와이파이 모양)을 누르고 TV를 선택하면 끝입니다.

내 폰 로그인 상태를 그대로 따라가기 때문에 구독 목록, 알고리즘도 그대로고, 유튜브 프리미엄 혜택(광고 제거)도 TV에서 그대로 적용됩니다.

미러링이랑 헷갈릴 수 있는데, 캐스팅은 TV가 영상 주소를 직접 받아와서 재생하는 방식이라 폰으로 다른 작업을 하거나 화면을 꺼도 영상이 끊기지 않습니다.

남의 기기에 내 계정을 직접 입력하기 찝찝할 때도 캐스팅은 TV에 아무 흔적을 남기지 않아서 보안 면에서도 제일 낫습니다.

방법 2. TV 코드로 연결 (캐스팅 안 될 때)

같은 와이파이인데도 TV가 캐스팅 목록에 안 뜰 때 쓰는 방법입니다.

TV 유튜브 앱 설정에서 TV 코드로 연결을 누르면 숫자 코드가 뜹니다. 폰 유튜브 앱에서 하단 마이페이지 메뉴 > 설정 > 동영상 및 오디오 환경설정 > TV로 시청하기 > TV 코드 입력 순서로 연결하면 됩니다.

로그인 오류를 완전히 무시하고 폰과 TV를 1:1로 강제 매칭하는 방식이라 튕김 문제가 있어도 연결됩니다.

집이나 장기 거주하는 곳

방법 1. youtube.com/activate 코드 연결

매번 캐스팅하기 번거로우면 처음 한 번만 제대로 연결해두는 게 낫습니다.

TV 유튜브 앱 로그인 화면에서 뜨는 코드를 휴대폰이나 PC 브라우저에서 youtube.com/activate에 입력하면 연결됩니다. 리모컨으로 직접 ID/비밀번호 입력하는 것보다 안정적이고, 어떤 채널로 연동할지 직접 선택할 수 있어서 채널 충돌로 인한 튕김도 없습니다.

방법 2. 모든 기기에서 로그아웃 후 재연결 (로그인이 꼬였을 때)

이미 로그인 기록이 꼬인 상태라면 TV 유튜브 앱 설정에서 모든 기기에서 로그아웃을 눌러 기록을 정리한 뒤, youtube.com/activate 방식으로 다시 연결해보세요.

방법 3. 휴대폰 구글 보안 알림 확인하기

TV에서 로그인을 시도하면 휴대폰으로 새 기기 로그인 승인 알림이 오는 경우가 있습니다. 이 알림을 확인하지 않으면 TV 쪽에서는 인증이 끝나지 않은 상태라 계속 로그인 오류가 반복됩니다.

로그인 튕김이 계속되면 휴대폰 알림창, 지메일, 구글 계정 보안 알림까지 같이 확인해보세요.

2단계 인증을 켜둔 계정이라면 이 부분이 특히 중요합니다.

방법 4. TV 유튜브 앱 캐시 삭제

TV나 크롬캐스트 쪽 유튜브 앱 데이터가 꼬이면 로그인 정보가 제대로 갱신되지 않아서 튕김이 생길 수 있습니다.

설정 > 앱 > YouTube > 캐시 삭제

이후 다시 코드 연결 방식으로 로그인해보세요.

방법 5. TV 시스템 앱 업데이트

유튜브 앱 자체는 최신인데 TV 백그라운드에서 화면과 통신을 담당하는 시스템 앱이 구버전이면 로그인이 튕기는 경우가 있습니다.

스마트 TV(안드로이드 TV 기준)라면 TV 리모컨으로 구글 플레이스토어에 들어가서 유튜브 앱뿐 아니라 Android System WebView와 Chromecast built-in 항목도 찾아서 최신 버전으로 업데이트해보세요.

크롬캐스트는 자동 업데이트라 따로 할 수 없습니다. 구글홈 앱에서 기기 재시작 정도가 최선입니다.

왜 이런 일이 생길까요?

기존 로그인 기록 충돌

여러 사람이 로그인했다가 로그아웃을 안 하고 나가면 인증 토큰이 뒤섞입니다. 펜션, 숙소 TV에서 특히 자주 생기는 이유입니다.

구글 계정 보안 인증 미확인

TV 로그인 시 휴대폰 승인이 필요한데 확인을 안 한 경우입니다. 로그인했는데 계속 튕긴다면 휴대폰 알림부터 확인해보세요.

채널 선택 충돌

구글 계정에 채널이 여러 개 연결돼 있으면 TV가 어떤 채널로 인증해야 할지 혼란을 일으킵니다. 코드 연결 방식으로 채널을 직접 선택해주면 해결됩니다.

TV 시스템 앱 미업데이트

유튜브 앱은 최신인데 TV의 WebView 같은 시스템 앱이 구버전인 경우입니다.
