아이폰 문자 안옴, 혹시 긴 문자나 기프티콘, 사진이 안 오나요?

요새 중학생된 조카가 학원을 여러 개 다니느라 힘들어하는 것 같길래, 며칠 전에 치킨 기프티콘을 살짝 보냈거든요. 서프라이즈로요.

그런데 며칠이 지나도 아무 반응이 없는 거예요. 요즘 젠지들은 원래 리액션을 잘 안 하나 요놈 요놈 싶었는데, 이상해서 카톡으로 슬쩍 물어봤더니 못 받았다고 하더라구요?

이게 어디로 증발을 했담? 분명히 돈 내고 보냈는데요. 혹시 못 찾은 거 아니냐고 했더니, 자기도 평소에 종종 문자를 못 받은 게 있다고 해요. 그것도 한두 번이 아니라고요.

엥, 뭐지? 어이가 없어서 조카 아이폰을 직접 들여다봤더니 설정 문제더군요. 기프티콘이나 사진, 학교, 학원 단체 문자처럼 안 오던 게 있으셨다면 이 두 가지만 확인해 보세요.

아이폰 문자 안옴 한줄 결론:
아이폰에서 사진이나 장문 문자가 안 온다면, 먼저 셀룰러 데이터를 켜고 MMS 메시지 설정을 확인해보세요.

먼저 해볼 것 1. 셀룰러 데이터 켜기

사진 문자, 기프티콘, 장문 문자가 안 온다면 제일 먼저 셀룰러 데이터부터 확인해보세요.

설정 > 셀룰러 > 셀룰러 데이터 켬

여기서 꺼져 있었다면, 다시 켠 뒤 메시지 앱을 열어보세요. 밀려 있던 문자들이 한꺼번에 쏟아져 들어오는 경우가 있거든요.

사실 이게 좀 의아하게 느껴지실 수 있어요. 데이터는 껐어도 와이파이 잘 되는데 왜 문자가 안 오냐고요.

그런데 아이폰에서 사진이나 기프티콘, 긴 내용이 담긴 문자는 MMS라는 방식으로 전송되는데, 이 MMS는 와이파이가 아니라 셀룰러 데이터를 타고 들어와요. 와이파이가 아무리 빵빵해도 셀룰러 데이터가 꺼져 있으면 이런 문자들은 안 들어오는 거예요.

평소에 데이터 아끼려고 셀룰러 꺼두시는 분들, 카드 승인 문자나 카톡은 잘 오니까 별 문제 없겠지 싶으셨을 텐데요. 짧은 문자(SMS)는 셀룰러 데이터 없이도 오거든요.

그러니 기프티콘이나 학교 단체 문자만 콕 집어서 안 왔던 거예요.

먼저 해볼 것 2. MMS 메시지 설정 켜기

셀룰러 데이터를 켰는데도 여전히 안 온다면, 이번엔 MMS 설정을 확인해보세요.

설정 > 메시지 > MMS 메시지 켬

이 항목이 꺼져 있으면 사진 문자나 기프티콘이 아예 수신이 안 돼요. 조카 폰이 딱 이 상태였거든요. 켜져 있지 않다면 초록색으로 바꿔주세요.

설정을 바꾼 직후에는 바로 안 들어올 수 있으니, 잠깐 기다렸다가 메시지 앱을 다시 열어보세요. 그래도 감감무소식이면 아이폰을 한 번 껐다 켜보시고요.

한 가지 참고하실 게 있는데요. 통신사나 요금제에 따라 이 MMS 메시지 항목이 아예 안 보이는 경우도 있어요. 그럴 땐 아래 통신사 설정 업데이트 쪽을 먼저 확인해보세요.

애플 공식 안내에서도 그룹 MMS 메시지를 사용하려면 설정 > 메시지 > MMS 메시지를 켜라고 안내하고 있어요. MMS 메시지 항목이 보이지 않는다면 통신사나 요금제에서 MMS를 지원하는지 확인해야 합니다.

참고: Apple 공식 도움말 – iPhone 또는 iPad에서 그룹 문자 메시지 보내기

그래도 안 오면: 통신사 설정 업데이트 확인하기

설정 > 일반 > 정보

이 화면에 들어간 뒤 10~15초 정도 가만히 기다려보세요. 통신사 설정 업데이트가 필요한 상태라면 팝업이 떠요. 뜨면 업데이트 눌러주시고, 아무것도 안 뜨면 이미 최신 상태인 거니까 넘어가셔도 됩니다.

유심을 바꿨거나 알뜰폰으로 이동한 뒤부터 사진 문자가 안 들어오기 시작했다면 이 부분을 꼭 확인해보세요. 통신사 설정이 새 환경에 안 맞게 남아있는 경우가 있거든요.

업데이트 후에는 아이폰을 한 번 껐다 켜고, 메시지 앱을 다시 열어보세요.

마지막 수단: 네트워크 설정 재설정

여기까지 다 해봤는데도 안 된다면, 네트워크 설정 재설정을 시도해볼 수 있어요. 다만 이건 진짜 마지막에 해보시는 걸 권장해요. 저장해둔 와이파이 비밀번호랑 일부 설정이 초기화되거든요.

설정 > 일반 > 전송 또는 iPhone 재설정 > 재설정 > 네트워크 설정 재설정

기기 암호를 입력하면 재부팅되면서 네트워크 설정이 싹 초기화돼요. 재부팅 후 셀룰러 데이터 다시 켜져 있는지 확인하고, 메시지 앱 열어보세요.

이 단계까지 다 해봤는데도 안 온다면, 그건 아이폰 설정 문제가 아니라 통신사 쪽 MMS 수신 문제일 가능성이 높아요. 이때는 사용 중인 통신사 고객센터에 직접 문의하시는 게 가장 빠릅니다.

아이폰 문자 안옴
왜 사진, 장문 문자만 안 올까?

아이폰에서 문자가 다 똑같이 오는 것처럼 보이지만, 사실 방식이 달라요.

카드 승인 문자, 입출금 알림, 택배 도착 문자처럼 짧은 문자는 SMS예요. 이건 셀룰러 데이터 없이도 들어와요.

반면 사진이 들어간 문자, 기프티콘, 학교, 학원 단체 안내 문자처럼 용량이 있는 문자는 MMS로 처리돼요. 이게 셀룰러 데이터랑 MMS 설정 두 가지 다 필요한 거예요.

겉으로 보면 다 똑같은 문자 앱에서 오는 것 같지만, 뒤에서 타고 오는 길이 달라요. 그래서 카톡이나 짧은 문자는 잘 오는데 기프티콘이나 단체 문자만 안 오는 이상한 상황이 생기는 거예요.

아이폰 문자 안옴, 이 글과 다른 경우라면

이 글은 짧은 문자는 오는데 사진, 기프티콘, 장문, 단체 문자만 안 오는 경우를 기준으로 정리했어요.

아래 상황은 원인이 달라요.

  • 인증번호 문자만 안 오는 경우
  • 특정 사람 문자만 안 오는 경우
  • 아이패드나 애플워치에는 오는데 아이폰에는 안 보이는 경우
  • 아이폰에서 갤럭시로 바꾼 뒤 문자가 안 오는 경우

이런 경우는 MMS 문제가 아니라 메시지 필터, 차단 설정, iMessage, 기기 동기화 쪽 문제일 수 있어요. 각각 다른 글에서 따로 다룰게요.

아이폰 문자 안옴 현상, 마치며

저희 조카 경우는 두 가지가 겹쳐 있었어요. 학생이다 보니 데이터 제한이 있어서 와이파이 되는 곳에서는 셀룰러 데이터를 꺼두는 버릇이 있었고, 거기다 MMS 설정까지 꺼져 있었던 거예요. 둘 다 켜주고 나서야 밀려 있던 문자들이 들어왔어요. 치킨 기프티콘도 무사히 도착했고요.

사진 문자나 기프티콘이 안 온다고 하면 일단 이 두 가지부터 확인해보세요. 생각보다 단순한 설정 문제인 경우가 많습니다.
